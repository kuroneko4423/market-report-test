# 株式市場予測レポート 2026年6月25日

## エグゼクティブサマリー

2026年6月24日（火）の日本株は米半導体株安を受けたリスク回避売りで一時1,300円超安と乱高下し、大引けは613円安となった。しかし翌25日未明にホルムズ海峡に足止めされていた船舶が相次いで脱出したとの報道で原油が一時70ドル割れ（3カ月ぶり）まで急落し、リスクセンチメントが改善。NYダウは182ドル高で反発し、大阪取引所夜間先物は+1,500円の71,060円まで上昇した。本日6月25日（木）の東京市場は原油安・NY株高を受けて大幅反発スタートが見込まれる。一方でウォーシュFRB議長のタカ派姿勢やナスダックの軟調継続、半導体株の不透明感が上値を抑える可能性がある。

## 2. 本日（6月25日）の市場予測

### 日本株式市場（東京市場）

**予測：大幅反発。ただし半導体株の動向次第で上値に不透明感**

| 指標 | 予測レンジ |
|------|----------|
| 日経平均 | 70,000〜71,500円程度 |
| 傾向 | 原油安・NY株高・先物高を受けて寄り付きから大幅高スタート |

**上昇要因：**
- 大阪取引所夜間先物：71,060円（前日比+1,500円）と強いシグナル
- NYダウ182ドル高・原油急落によるリスクオン回帰
- ホルムズ海峡の緊張緩和→エネルギーコスト低下→内需・輸送セクターにプラス
- 個人信用買いの含み益が13年ぶり水準→個人投資家のマインド改善
- FRBストレステスト全行通過→金融株への安心感

**下落・上値抑制要因：**
- ナスダックはマイナス圏継続→半導体・AI関連株の重石
- ウォーシュFRB議長のタカ派姿勢（インフレに「厳しい姿勢」）→米利下げ期待後退
- 中国のタングステン禁輸継続（AI向け半導体の材料懸念、特殊ガス株は急騰も不透明感残る）
- 日経平均VI高水準継続による値動きの不安定さ
- 米新築住宅販売5月が予想外の減少（景気の不透明感）

**注目セクター：**

| セクター | 方向性 | 材料 |
|---------|--------|------|
| 石油・資源 | 下落 | 原油急落（3.92%安）が直撃 |
| 航空・輸送 | 上昇 | 原油安でコスト軽減期待 |
| 素材（特殊ガス） | 注目 | 中国タングステン禁輸で急騰継続の可能性 |
| 半導体・AI | 上値重い | ナスダック軟調・SKハイニックス動向 |
| 金融・銀行 | 上昇 | FRBストレステスト全行通過・利上げ観測 |
| 電力・エネルギー | 混在 | 原油安はプラス、再エネ移行コストは変わらず |
| 自動車 | 上昇 | ホンダ・日産・三菱自が中核部品共通化でコスト低減期待。トヨタは米国販売首位返り咲き視野 |

## 3. 主要リスク要因

### 上方リスク
1. **マイクロン決算の好内容**：半導体セクター全体への波及効果で、日経平均が71,500円超えを狙う展開も
2. **ホルムズ海峡の完全安定化**：原油のさらなる下落→エネルギーコスト低下→消費・輸送株に追い風
3. **日本の成長戦略**：官民投資370兆円（2040年度まで）の具体化が進めば中長期的な押し上げ材料

### 下方リスク
1. **マイクロン決算が失望**：半導体株全体に売りが広がれば日経平均が70,000円を割り込む可能性
2. **ウォーシュFRB議長の追加タカ派発言**：利上げ観測強まれば株価と債券の同時下落リスク
3. **ホルムズ海峡の緊張再燃**：イラン・オマーン間の調整は続いているが不確実性は残存
4. **為替介入**：ドル円が161円台後半に迫れば政府・日銀の口頭・実弾介入リスク浮上
5. **中国タングステン禁輸の拡大**：AI・半導体サプライチェーン全体への影響拡大懸念

## 4. 注目ニュース・材料

### 日本国内
- **NTT外債発行**：1.6兆円規模（アジア企業で今年最大）の外債発行へ。信用市場の底堅さを示す
- **SKハイニックス ナスダック上場申請**：約4.8兆円調達計画。韓国半導体大手の動向が東京市場にも波及
- **ソフトバンク・東京電力HD出資**：孫氏がAIデータセンターを意識した出資意欲。電力インフラ・AI関連株に注目
- **仮想通貨投資（DAT企業）**：TOPIX組み入れをめぐる論争（TOPIXは除外方針）。クリプト関連株の行方
- **日本政府の外貨準備運用改善検討**：成長戦略草案に言及。中長期的な円相場への影響注視
- **NTN・パナソニック等、関西企業の経営改革加速**：中小型株・バリュー株への物色機運

### 海外
- **FRBストレステスト通過（32行全て）**：自社株買い・増配解禁→金融株に追い風
- **JPモルガン S&P500年末目標7,800**：強気見通しが米国株全体のセンチメントを支援
- **ドイツ企業見通し改善**：不確実性緩和でIFO指数が上昇、欧州景気の底打ち期待
- **SKハイニックス ナスダック上場申請**：約4.8兆円調達計画で半導体セクターの資金需要に注目
- **トヨタ 米国販売首位返り咲き視野**：ハイブリッド好調で増勢、米国市場での競争力再確認
- **ホルムズ海峡 船舶脱出・航行再開**：イランとオマーンの協議が進展、地政学リスクプレミアム縮小
