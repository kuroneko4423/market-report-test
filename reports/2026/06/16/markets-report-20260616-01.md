# 本日の株式市場予測レポート（2026年6月16日）

## エグゼクティブサマリー

**米イラン和平合意**を主因とするリスクオン相場が続く中、本日の日本・米国株式市場は引き続き強い上昇基調が見込まれる。前日（6月15日）の米国市場でダウ工業平均が最高値を更新し、日経平均先物も夜間取引で大幅高を維持していることから、日本市場は寄り付きから強気な展開が予想される。一方、原油価格の急落が関連セクターに下押し圧力をもたらす点には注意が必要。

| 指数 | 動向 | 備考 |
|------|------|------|
| ダウ工業平均 | **最高値更新**（+468～+596ドル） | 米イラン合意でリスクオン |
| NASDAQ | **大幅高**（+2.9%） | 半導体・AI関連に買い集中 |
| S&P500 | 上昇 | 全面高の様相 |

**主な買い材料**
- 米国とイランの和平合意（枠組み合意）が成立し、中東地政学リスクが急速に後退
- 原油価格急落によるインフレ圧力緩和への期待が高まった
- スペースX株が上場2日目に約20%高、時価総額が400兆円を突破しセンチメントを押し上げ

- 米10年国債利回りは**4.47%**に低下（長期債反発）
- 原油下落によるインフレ懸念後退が長期金利の低下を促した

### 為替市場
- **ドル/円**：ドルが小幅安、円は若干上昇
- 米イラン合意を背景にリスクオン心理がドルを押し下げ
- 複数の主要中央銀行会合（ECB等）を控え、投資家は様子見姿勢も混在

### 商品市場
- **原油（WTI）**：一時80ドル割れ、5%超安（ブレント）
  - ホルムズ海峡の緊張緩和・イラン原油供給再開期待
  - JPモルガンは合意進展次第で**70ドルまで下落**の可能性を示唆
- **金**：続伸（安全資産需要は後退するも底堅い）

## 3. 本日（6月16日）の市場予測

### 日本株式市場

**予測：強気（大幅続伸の公算大）**

| 項目 | 予測 |
|------|------|
| 日経平均（寄り付き） | 前日比プラス圏スタート |
| 日経平均（大引け） | **69,500円～70,000円台**を視野 |
| 基調 | リスクオン継続、米最高値追随 |

**根拠**
- 日経平均先物（大取夜間）が**69,840円**水準で推移（+約3.2%超高）
- 前日NY市場でダウが過去最高値更新、NASDAQ+2.9%と強烈なポジティブサプライズ
- 米イラン和平合意のリスクオン余韻が継続
- 円が小幅安推移で輸出企業の業績期待が支え

**注目セクター**
- **強気**：半導体・電子部品（AI需要継続）、輸出関連（円安傾向）、非鉄金属・銅関連（銅価格高）、宇宙・防衛関連
- **弱気**：石油・エネルギー関連（原油急落の影響）、石化関連
- **中立**：銀行・金融（日銀政策据え置き見通し）

### 米国株式市場

**予測：強気（上昇継続も利益確定売りに注意）**

| 項目 | 予測 |
|------|------|
| ダウ工業平均 | 最高値圏でのもみ合いないし小幅続伸 |
| NASDAQ | AI・半導体主導で高値圏維持 |
| 基調 | 強気継続、ただし一部利確売り出現の可能性 |

**根拠**
- 米イラン合意によるエネルギーコスト低下が企業収益改善期待に直結
- エヌビディア・スペースX・セールスフォースなど個別銘柄の強材料が継続
- フォックスによるロク（Roku）の220億ドル買収合意で、メディア・エンタメセクターにも注目
- ただし、NAHB住宅市場指数（6月35）が予想外に悪化、製造業生産の失速（今年初）など弱い経済指標が若干の重しに

**リスク要因**
- 原油急落が想定以上に進みエネルギーセクターが重荷
- 米イラン合意の詳細が不透明で、地政学リスクの再浮上懸念
- ECB・英中銀等、各国中銀の金融政策スタンスへの警戒感（ECBラガルド総裁はエネルギー価格上昇の波及を指摘）
- 住宅・製造業の弱さが景気先行きへの不安を引き起こす可能性

## 6. 総合評価

```
日本株式市場  ：★★★★★（強気）
米国株式市場  ：★★★★☆（やや強気）
ドル/円      ：円高方向（リスクオン後退とエネルギー安が支え）
原油         ：下落基調継続
```

**総評**：米イラン和平合意という歴史的な地政学リスク解消が相場の最大の押し上げ要因。日経平均の70,000円台乗せが現実的な視野に入ってきた。ただし、米国の住宅・製造業の弱さや、エネルギー株の重荷が指数の上値を抑える可能性もある。半導体・AI関連・輸出株を中心にリスクオン継続を見込むが、原油価格の下落スピード次第では一部セクターに利益確定売りが出やすい。本日の日本株は寄り付きから強い展開が予想され、節目の70,000円を意識した値動きになる見込み。
