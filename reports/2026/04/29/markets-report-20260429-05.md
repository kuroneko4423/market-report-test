## 1. 前日（4月28日）の市場動向

### 日本市場

- **日経平均株価**: 3日ぶり反落、**619円安**。半導体・ハイテク主力株に利益確定売りが集まり、日銀会合の結果も重荷となった。

- **為替**: ドル/円は159円前半で上値重く推移。日銀会合後に一時的な円高圧力。
- **セクター動向**: 銀行株が強く、三井住友FGは5%高。証券会社各社も2026年3月期の純利益増加（大手3社最高益）を発表。一方、化学・半導体関連は軟調。
- **主要決算**: NEC（純利益54%増）、三菱電機（16%増益）、TDK（3年連続最高益）、三菱UFJ証券（20%増）、SMBC日興（75%増）など強い決算が相次いだ。

- **ナスダック**: 約1.3%の大幅安。AI銘柄に売りが集中（「マスクVSアルトマン」裁判開始もセンチメントを悪化）。

## 3. 日本株式市場の予測

### 全体見通し：**中立〜やや軟調**（底堅い動きを想定）

シカゴ日経平均先物（59,210円）が示す通り、前日の大幅安からの自律反発余地があるが、AI・半導体セクターの重しとFOMC前の様子見ムードが上値を抑える。

#### 上昇要因

1. **銀行・金融株の続伸期待**: 日銀の「早期利上げ示唆」を背景に、三井住友FG・三菱UFJ等のメガバンクに引き続き買いが集まりやすい。
2. **日本株ADRの買い優勢**: 前日の米国市場でオリックス、三井住友FGなど日本株ADRが高い。
3. **好業績銘柄への評価**: NEC、三菱電機、TDKなど強い決算を発表した銘柄の個別物色が継続。
4. **自社株買い**: デンソー（最大3,136億円のTOB）、富士電機（最大210億円）など株主還元強化が評価される。
5. **原油関連・エネルギー株**: 原油100ドル超で石油・エネルギー関連は恩恵。

#### 下落要因

1. **半導体・ハイテク株の軟調継続**: OpenAI問題に端を発するAI投資懸念が消えず、東京エレクトロンなど半導体関連への売り圧力が続く恐れ。
2. **中東リスクによるナフサ高騰**: デンソー・豊田合成など自動車部品各社が「6月からナフサコスト増を懸念」と表明。化学・素材株の圧迫要因。
3. **円高リスク**: 日銀タカ派受け止めによる円高圧力（ドル/円159円台）が輸出株の重石。
4. **FOMC前の様子見**: 本日夜から開始のFOMCを前に、積極的な売買を手控える動きが予想される。

## 4. 米国株式市場の予測

### 全体見通し：**中立〜慎重**（FOMC待ちの様子見）

FOMC（4月29〜30日）の開催を前に積極的なポジション拡大は見込みにくい。AI株の売り圧力が継続する一方、消費財（コカ・コーラ）・自動車（GM）の好業績や消費者信頼感の改善がダウの下支えとなる。

#### 上昇要因

1. **消費者信頼感の改善**: 4月CB消費者信頼感が92.8に予想外の上昇。個人消費の底堅さを示唆。
2. **ディフェンシブ株・好業績銘柄**: コカ・コーラ（増収増益）、GM（Q1増益・通期上方修正）などが市場を支える。
3. **ゴールドマンの「押し目買い」推奨**: 短期の下落シグナルを押し目と判断する機関投資家の動きが下値を支える可能性。
4. **エネルギー株**: 原油高が継続する中、石油メジャーへの資金流入が続く。

#### 下落要因

1. **AI・ハイテク株の軟調継続**: OpenAI問題・マスク訴訟でAI投資の先行き懸念が払拭されていない。
2. **スタグフレーションリスク**: GMやブラックロックが警告するインフレの「広範囲」への影響、利益圧迫懸念。
3. **中東情勢の長期化**: イランとのホルムズ海峡を巡る交渉が難航。ホルムズ再開への道筋が不透明なまま。
4. **ダイモン氏の警告**: JPモルガンCEOがクレジット市場低迷のリスクを指摘。金融システムへの懸念。
5. **UAEのOPEC脱退**: 原油相場の先行き不安定化。
6. **トランプ大統領の支持率低下**: 34%（現任期最低）。政策の不透明感が続く。

| 生活必需品 | ↑ 強気 | コカ・コーラ等の好業績 |
| 自動車 | ↑ 中立〜強気 | GM好業績、ただしインフレ懸念 |
| テクノロジー | ↓ 弱気 | AI懸念持続（FOMC待ち） |
| 半導体 | ↓ 弱気 | OpenAI問題、AI投資縮小懸念 |
| 金融 | → 中立 | FOMC前の様子見 |
| 公益 | → 中立 | 金利動向次第 |
