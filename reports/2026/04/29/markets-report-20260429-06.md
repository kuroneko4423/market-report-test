## 主要テーマ分析

### 1. 中東情勢（最大リスク要因）

中東の地政学的緊張が引き続き市場の最大リスクとなっています。

| 項目 | 内容 |
|------|------|
| UAEのOPEC脱退 | 5月1日付けで脱退決定、石油生産を段階的に引き上げへ（OPEC内の結束に亀裂） |
| 米・イラン交渉 | 停滞が続く。トランプ大統領「イランは崩壊状態」と発言、ホルムズ海峡開放を要請 |
| ホルムズ海峡 | 日本の出光丸がサウジ産原油を積んで通過。通航リスクは継続 |
| アラムコ | LPG出荷停止を5月も継続、供給逼迫が深刻化 |
| ユーロ圏インフレ期待 | 1年先が4%に大幅上昇（イラン戦争の影響） |

**日本への影響：**
- 中東からのナフサ輸入が4割減少（3月貿易統計）
- デンソー・豊田合成など自動車部品サプライチェーンへの影響拡大
- 中部電力が今期業績見通しを「未定」に変更（中東リスクで不確実性高まり）
- 銭湯業界：重油高騰で経営危機

### 2. 日銀の金融政策正常化

今回の会合で3委員が利上げに賛成票を投じたことは市場に強いシグナルを送りました。

**注目ポイント：**
- 物価見通し（26年度）を+1.9%から**+2.8%へ大幅引き上げ**は次回利上げへの地ならし
- 植田総裁は「原油上昇が財・サービスの値上げに波及する可能性」に言及
- 日銀は「ビハインド・ザ・カーブ」回避を明言 → 追加利上げへの積極姿勢

**恩恵セクター（銀行・保険）：**
- 三菱UFJ証券2026年3月期純利益+20%、SMBC日興純利益+75%（最高益）
- 大手証券3社が最高益更新（野村・大和・三菱UFJ系）
- 銀行株の継続的な上昇が見込まれる

### 3. AI・半導体セクターの調整リスク

- **OpenAIの売上高目標未達報道** → ハイテク株全面安のトリガーに
- OpenAI自身は「フル稼働」と反論しているが、市場の懐疑感は払拭されず
- ナスダックが主要指数をアンダーパフォーム（-1.3%）
- 中国の曦智科技（AI関連）が香港上場初日に400%高 → AI熱自体は継続
- 日本国内では半導体主力株（東京エレクトロン等）が引き続き重い展開

### 4. FRB・FOMC動向（最重要イベント）

- FRBは**3会合連続で政策金利を据え置く見通し**（Bloomberg・Reuters報道）
- 議長交代を巡る政治的な動きにより「FOMC自体は影が薄い」状況
- 米4月消費者信頼感指数：92.8（予想外の上昇）→ 景況感は底堅い
- 米住宅価格指数（2月）：前月比横ばい
- **FOMC結果は4月29日の米国市場の方向性を決定する**

### 5. 中国経済・人民元動向

- ムーディーズが中国の格付け見通しを「安定的」に引き上げ
- 中国人民銀が一部銀行に4月の融資拡大を指示（信用急減速への警戒）
- 人民元高による中国企業の為替差損が利益を圧迫 → 本土株の持ち直しに逆風
- 中国共産党がエネルギー安全保障の強化方針を発表

## まとめ

2026年4月29日（昭和の日）は東京株式市場が休場のため、国内投資家の注目は**FOMC政策金利決定（4月29日米国時間）**に集まります。

市場の大きなテーマは以下3点：

1. **日銀のタカ派転換**：3名の利上げ賛成・物価見通し大幅引き上げ → 銀行・金融株が恩恵、成長株・グロース株に重石
2. **中東・原油情勢**：原油の2週ぶり高値更新・UAEのOPEC脱退・ホルムズリスク継続 → エネルギー株恩恵、化学・自動車に下押し
3. **AI・ハイテク調整**：OpenAI懸念でナスダック軟調 → 日本の半導体・AI関連株にも影響

4月30日の東京市場再開時には、29日の米国市場（特にFOMC結果と主要指数の動向）と為替水準が最初の値動きを決定する重要な先行指標となります。引き続き銀行セクターと原油関連に注目しつつ、半導体・AI関連はFOMC通過後の反発タイミングを慎重に見計らう局面です。
