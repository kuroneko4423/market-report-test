# 株式市場予測レポート（2026年4月29日）

## 概要

本レポートは、2026年4月29日（水）の日本・米国株式市場の動向予測をまとめたものです。前日（4月28日）の各市場の動きと、本日に影響を与える主要なニュースを分析し、本日の市場見通しを提示します。

## 1. 前日（4月28日）の市場動向

### 日本株式市場

- **日経平均株価**：前日比619円安で引けた（3日ぶり反落）。前場では半導体株を中心に利益確定売りが進み、後場は日銀金融政策決定会合の結果公表後に円高が進展し下げが加速した。
- **日経平均先物**（夜間取引）：1,010円安の59,010円で終了。本日の寄り付きに下押し圧力となる見込み。
- **日本株ADR**（28日）：買い優勢。オリックス・三井住友フィナンシャルグループが上昇。
- **銀行株**：早期利上げ期待の高まりから三井住友FGが5%高など大幅上昇。
- **半導体関連株**：米OpenAI収益目標未達観測でNYダウが下落したことが波及し、売り優勢となった。

### 米国株式市場

- **NYダウ**：続落。OpenAIが収益目標を達成できないとの観測報道から半導体関連・AI関連銘柄が全面安となった。
- **ナスダック**：1.3%の大幅安。IT・ハイテク株への売りが集中。
- **米7年債入札**：最高落札利回り4.175%。
- **NY債券**：10年債利回り4.34%（原油高を受け一時3週ぶり高水準）。
- **米コカ・コーラ**：第1四半期は増収増益（新CEOは「力強いスタート」と表明）。
- **米UPS**：26年通期売上高見通しを維持。燃料急騰による需要圧迫リスクを指摘。
- **米4月CB消費者信頼感指数**：92.8に改善（見通し改善が寄与）。

### 為替・原油・金

- **ドル円**：1ドル＝159円台後半（原油高でドル買い優勢、日銀会合後の円高は一服）。
- **NY原油**：約3.69%高。UAE（アラブ首長国連邦）の5月1日からのOPEC脱退発表、及びイラン・ホルムズ海峡を巡る地政学的緊張が続き、2週間ぶりの高値圏。
- **NY金**：1.82%安（原油高・ドル高が重し）。

## 2. 本日の主要注目ポイント

### 日銀金融政策決定会合の影響

4月28日の日銀会合では、政策金利は予想通り据え置きとなったものの、委員3名が反対票を投じる「タカ派サプライズ」となった。植田総裁は会見で以下を示唆：

- 「原油上昇が財・サービスの値上げに従来以上に波及している」
- 「経済は下振れリスク、物価は上振れリスクが大きい」
- 「ビハインド・ザ・カーブに陥らないよう次回以降適切に対応」
- 「26年度中心に物価見通しを上方修正（2026年度：2.8%）」

→ **早期利上げへの市場の期待が強まりやすい状況**。銀行・金融株には引き続き追い風となる一方、成長株・ハイテク株には逆風。

### 米FOMCへの注目（4月29日〜30日開催）

FRBは3会合連続の金利据え置きが見込まれている。議長交代を巡る政治的ドラマの影響があるものの、基本的にはハト派方向への転換期待は薄く、**高金利・インフレ懸念継続の環境**が続く。FOMCの声明・パウエル議長会見が夜間に市場に影響を与える可能性がある。

### 地政学リスク：イラン・ホルムズ海峡

- UAEが5月1日付でOPECを脱退（生産を独自に段階的引き上げ方針）。
- イランのホルムズ封鎖は現在も緊張状態。トランプ大統領は「イランは封鎖解除を要請してきた」と主張するも、交渉は膠着。
- 欧州ガス先物：LNGタンカーが戦争以来初めてホルムズを通過したことで1%下落。
- トヨタ系部品（デンソー、豊田合成など）：「ナフサの6月からの供給懸念」を表明、自動車サプライチェーンへの影響が懸念される。

### OpenAI・AI関連株への圧力

OpenAIの収益目標未達観測がAI・半導体株への売り材料となっている。マスク対アルトマン裁判が開始され、OpenAI営利化の妥当性が争われていることも不透明感を増幅。**ナスダック中心に売り圧力が継続するリスク**がある。

## 3. 本日の株式市場予測

### 日本株式市場

| 項目 | 予測 |
|------|------|
| **市場全体の方向性** | 下落（上値重い展開） |
| **日経平均の目安レンジ** | 59,000〜60,200円程度 |
| **主要テーマ** | 日銀利上げ警戒・半導体安・原油高 |

**根拠：**

1. **下押し要因**
   - 日経平均先物が夜間で1,010円安の59,010円で終了しており、寄り付きは大幅安スタートが見込まれる。
   - 米国株がAI・半導体株全面安で終わり、東京市場でも値がさ半導体株に売り圧力。
   - 原油高による輸入コスト増懸念（自動車・化学・航空など幅広いセクターへの影響）。
   - ドル円が159円台後半と円安水準にあるため輸入インフレ圧力が意識される。

2. **下支え・反発要因**
   - 銀行株は引き続き「利上げ観測」で買い。三菱UFJ・三井住友FG・みずほなど大手行が底堅い動き。
   - ADRではオリックスや三井住友FGが高く、一部個別株はプラスの動きが見込める。
   - 証券会社の好決算（大手証券の2026年3月期純利益17%増など）がポジティブ。
   - TDK・三菱電機など好業績銘柄は個別で買いが入りやすい。

3. **セクター別見通し**
   - **上昇期待**：銀行・証券・保険（利上げ観測）、エネルギー・石油（原油高）
   - **下落懸念**：半導体・電子部品（AI株安の波及）、自動車（ナフサ・原油高によるコスト増）、小売・消費財（輸入コスト増）

### 米国株式市場

| 項目 | 予測 |
|------|------|
| **市場全体の方向性** | 不安定（FOMC待ちで様子見・やや下落バイアス） |
| **NYダウの動き** | もみ合い～小幅安 |
| **ナスダックの動き** | 引き続き軟調（AI関連株への不透明感） |

**根拠：**

1. **下押し要因**
   - OpenAI収益懸念に加え、マスク対アルトマンの裁判がAI投資全般への懐疑心を高める。
   - 原油高がインフレ再燃懸念を強め、FRBの利下げ転換期待を後退させる。
   - ユーロ圏インフレ期待が1年先4%まで上昇し、グローバルなインフレリスクを示唆。
   - ブラックロックは「国債利回り高止まり継続」を予想（リスク資産から安全資産へのシフト）。

2. **下支え要因**
   - 米4月CB消費者信頼感指数が92.8に改善（市場予想を上回る）。
   - コカ・コーラ、スターバックス（9四半期ぶり増収増益）など消費財関連の好決算。
   - FOMC声明が予想通りの据え置きであれば、一旦の安心感から反発する場面も。

3. **FOMC後のシナリオ**
   - **据え置き・ニュートラルな声明**：市場は中立的反応、短期的安心感から小幅反発の可能性。
   - **タカ派な声明（インフレ警戒強調）**：株安・ドル高・長期金利上昇の連鎖が起きうる。

## 4. 為替・商品市場の見通し

### 為替（ドル円）

- 原油高・FOMC待ちの環境でドル買い優勢が継続し、**159〜161円のレンジ**で推移する可能性。
- 日銀の早期利上げ観測が高まれば、円買いが再燃して一時的に円高方向に振れる場面もあり得る。

### 原油

- UAEのOPEC脱退は短期的には供給増懸念として原油安要因だが、ホルムズ海峡の地政学的リスクが勝り、**引き続き100ドル前後の高水準**が見込まれる。
- イランとの交渉が進展すれば急落リスクがある一方、協議の膠着が続けばさらなる上昇圧力も。

### 金

- 原油高・ドル高の環境下では売り圧力。ただし地政学リスクが高まれば安全資産として買われる局面もあり、**方向感が出づらい**展開。

## 6. リスクシナリオ

### 上振れリスク

- 米FOMC後に「次回利下げを示唆」する文言が盛り込まれた場合→株高・円高
- イラン・米国間の外交交渉が急進展し、ホルムズ海峡の緊張が緩和→原油急落・株高

### 下振れリスク

- FOMCがインフレ長期化リスクを強調し、タカ派姿勢を強化した場合→株安・円高
- OpenAI関連の悪材料が継続し、AI・半導体セクター全体への売りが波及
- ホルムズ海峡でのさらなる事案（追加封鎖・軍事衝突）→原油急騰・株急落
