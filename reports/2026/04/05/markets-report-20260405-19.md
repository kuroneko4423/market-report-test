# 株式市場動向予測レポート（2026年4月5日）

## 概要

本レポートは、2026年4月5日時点の主要ニュースを基に、本日および週明けの日本・米国株式市場の動向を予測するものです。

---

## 主要ニュースサマリー

### 1. 中東情勢（最重要リスク要因）

- **イランが米軍戦闘機を撃墜**し、湾岸諸国のエネルギー施設への攻撃が拡大（ブルームバーグ）
- トランプ米大統領は「あと48時間」とイランに最後通牒を突きつけ、橋・発電所の破壊を警告（みんかぶ）
- **イランはホルムズ海峡の長期閉鎖**を宣言。同海峡を通る石油・LNGの輸送リスクが高まっている（みんかぶ）
- 商船三井のLPG船がホルムズ海峡を通過（日本関係2隻目）。日本のエネルギー輸入への影響が懸念される（ロイター）
- フランス大統領は「武力による海峡再開は非現実的」と発言し、紛争の長期化リスクを示唆（みんかぶ）
- イタリア首相がサウジ皇太子と会談し、ホルムズ問題を議論（日経速報）

### 2. 原油・エネルギー市場

- **NY原油が111ドル台に急騰**（前日比+11.41%の大幅上昇）。ホルムズ海峡閉鎖リスクを背景に買いが加速（みんかぶ）
- ドバイ原油も3日午後に上昇（日経マーケット）
- 日本政府はナフサ「国内需要4カ月分確保」を発表（高市首相がXに投稿）。石油節約策も検討中（日経）
- 赤沢経産相が「中東依存物資の実態把握」を指示（日経）
- 原油高が企業業績に波及する懸念から、小売・飲食・輸送関連企業の見通しが悪化しつつある

### 3. 為替市場

- ドル円は**159円台後半**で推移。グッドフライデーで欧米は休場の中、落ち着いた動き（みんかぶ）
- **160円を試す動き**が継続。介入警戒感が支えとなっている（みんかぶ）
- 片山財務相は「為替の動きは石油市場に引きずられている」と発言し、「法的に可能なことはすべてあり得る」と介入可能性を示唆（ロイター）
- 金融機関の円相場見通しでは、6月末に150円台後半への円高修正を予想する声もある（日経）

### 4. 米国経済指標・金融政策

- **3月雇用統計：+17.8万人増加**（予想上回る）、失業率4.3%に低下（日経・ブルームバーグ）
- 米国債利回り：10年債4.34%（雇用統計後に上昇後、イラン関連ニュースで小幅低下）（みんかぶ）
- **米サービス業PMIが縮小圏**（2023年以来）。イラン戦争で新規受注が低下（ブルームバーグ）
- FRB利下げ観測が後退。SF連銀総裁は「雇用の伸びゼロでも労働市場が弱いとは限らない」と発言（ブルームバーグ）
- **ECBは利上げか据え置きかで議論**へ（オランダ中銀総裁）

### 5. 日本株・日本経済

- 本日（4/5）東京株式は**大引けで660円高**と反発。半導体などハイテク株中心に買い流入（みんかぶ）
- 日経平均先物（夜間）は70円高の53,270円（日経）
- 連合の賃上げ率は平均**5.09%**（第3回集計）。中小企業も5%台維持（日経）
- JPXがコーポレートガバナンス指針を5年ぶりに改訂。現預金の有効活用を企業に促す（日経）
- 自社株買いの設定件数が東証要請以降で初の前年度割れ（ブルームバーグ）
- 1〜3月の企業年金利回りは国内株式堅調で4四半期連続プラス（日経）

## 市場動向予測

### 日本株式市場（日経平均）

**予測：上値重く、神経質な展開。レンジ：52,500〜53,800円程度**

| 堅調な賃上げ | 内需回復への期待感（連合5.09%） |

#### 弱気材料

| 材料 | 内容 |

| ホルムズ閉鎖リスク | エネルギー輸入コスト増大の恐れ |
| 日銀利上げ観測 | 金融引き締めへの警戒感（160円攻防） |

| 自社株買い鈍化 | 需給の悪化懸念 |

#### 総合判断

本日の東京市場は前日の大幅安（トランプ演説による失望感と原油高）からの反発で660円高を達成。ただし、「底打つコツンの音はまだ遠く」（日経）との見方が示すように、持続的な上昇には中東情勢の落ち着きが不可欠。  
来週はイランへの48時間最後通牒の期限到来とその後の展開が相場の方向性を大きく左右する。エネルギー関連株は原油高の恩恵を受けやすい一方、航空・輸送・素材・小売関連はコスト増が重荷。防衛関連株は地政学リスク継続により引き続き物色される。

### 米国株式市場（NYダウ・S&P500・ナスダック）

**予測：下値警戒継続、不安定な展開。週明けは地政学リスクの消化待ち**

#### 強気材料

| 材料 | 内容 |
|------|------|
| 雇用統計堅調 | 3月+17.8万人増（景気底堅さ示す） |
| 失業率改善 | 4.3%への低下 |
| 恐怖指数（VIX）低下 | 市場の過度な警戒感は後退 |

#### 弱気材料

| 材料 | 内容 |
|------|------|
| 中東情勢悪化 | イランとの軍事的緊張が続く |
| 原油急騰 | エネルギーコスト上昇でインフレ再燃 |
| FRB利下げ観測後退 | 高金利継続が株式バリュエーションの重荷 |
| サービス業PMI悪化 | 2023年以来の縮小圏 |
| 米株先物が軟調 | 週末にかけて上値が重い |

#### 総合判断

グッドフライデー（4/4）は米国株式市場が休場。週明け（4/6）には中東情勢のさらなる展開を受けて開く可能性が高い。雇用統計が堅調でFRBの利下げ余地が限られる中、原油急騰によるインフレ再燃が最大の懸念材料。  
ダウ平均は前日比小幅安（111ドル安）で引けており、原油高への対応次第では週明けに売りが先行する場面も想定される。ただし、強い雇用統計はリセッション懸念を抑制するポジティブ材料でもあり、急落リスクは限定的か。

## まとめ

2026年4月5日の株式市場を取り巻く最大の変数は**中東（イラン）情勢**であることは明らかです。ホルムズ海峡の閉鎖リスクが原油価格を111ドル台に押し上げ、エネルギーコストの増大を通じて企業業績・インフレ・金融政策すべてに波及しています。  

日本株は本日反発しましたが、原油高・円安・日銀利上げ観測という三重の圧力下にあり、上値は限定的。米国株は強い雇用統計があるものの、サービス業PMIの悪化とFRB利下げ観測後退により、高値追いの相場展開とはなりにくい状況です。  

来週は**イランへの最後通牒の期限（48時間）後の展開**が最大の焦点となります。地政学的緊張が高まる局面では、エネルギー・防衛関連株が物色対象となり、輸送・小売・食品セクターへの下押し圧力が続くと予想されます。
