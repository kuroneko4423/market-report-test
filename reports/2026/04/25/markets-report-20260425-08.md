## 2. 主要テーマ分析

### 2.1 米国・イラン核交渉（最重要材料）
4月25日にトランプ政権のウィットコフ特使がパキスタンでイラン外相と直接協議を実施予定。
- **楽観シナリオ**：協議進展 → 原油供給増期待 → 原油安 → インフレ鈍化期待 → 株高・ドル安
- **悲観シナリオ**：交渉決裂 → 原油急騰再開 → コスト高懸念 → 株売り圧力
- 前日の原油相場は既に「協議期待」を織り込む形で反落しており、交渉結果次第で大きな価格変動が生じる。
- トランプ氏は国民に「しばらくの間」ガソリン高騰を覚悟するよう言及しており、交渉長期化リスクも残存。

### 2.2 半導体・AI関連の強地合い
- インテルの好決算見通しを受け、AI投資サイクルの継続が確認された格好。
- 中国DeepSeekが最新AIモデルのプレビュー版を公開し、AI技術開発競争が加速。
- アルファベットのアンソロピック追加投資報道もAIセクターに強気ムード。
- 日本市場でも半導体関連（ルネサス、キーエンス等）の堅調が継続見込み。

### 2.3 消費者心理の悪化
- **ミシガン大消費者信頼感指数（4月確報値）：49.8（過去最低）**
  - イラン戦争長期化によるインフレ懸念が主因
  - P&G：原油高による業績圧迫を発表（FY2027 約10億ドル下押し）
  - 消費関連株・小売株への警戒感が高まっている。

### 2.4 日本の金利動向
- 日本長期金利（10年債）が一時**2.445%**に上昇（インフレ懸念）
- 日銀利上げ議論：経済学者の間でも意見が割れる状態
  - 「円安是正のために利上げを」vs「景気下押しリスク」
- かんぽ生命：今年度の国内債券購入額を2兆円弱と発表（金利上昇局面での運用方針）
- 住友生命：国内債キャピタル益狙いはやや慎重スタンス
- 金利上昇はバリュー株・金融株に追い風、グロース株・ハイテク株には逆風となりうる。

### 2.5 地政学リスク全般
- **イスラエル・ヒズボラ**：停戦に否定的な発言が続き不安要因
- **ロシア**：中銀が0.5%利下げ（▼14.5%へ）、経済下支えを優先
- **中米関係**：独自規制として米国からの国内テック企業への投資制限検討、一方で中国からパンダ2頭を貸与（首脳会談前の友好演出）

## 4. 主要リスクシナリオ

### 上振れシナリオ（強気）
米イラン交渉が進展し、停戦・制裁緩和への期待が高まった場合：
- 原油価格が大幅下落 → エネルギーコスト低下期待
- リスクオン相場が加速 → 日米欧株全面高
- 円高進行に注意（ドル安圧力）

### 下振れシナリオ（弱気）
米イラン交渉が決裂・膠着し、原油価格が急反発した場合：
- コスト高 → 企業業績への懸念再燃
- インフレ懸念再燃 → 利上げ長期化観測
- 株全体に売り圧力、特に消費財・輸送セクター
