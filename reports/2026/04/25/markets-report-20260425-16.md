## 2. 本日の主要トピック

### 【最重要】米国・イラン和平協議の進展

トランプ大統領は4月25日、ウィットコフ特使らをパキスタンに派遣し、イラン外相と協議を行う見通し。トランプ大統領は「イランが米国の要求に応じる可能性」を示唆しており、中東情勢の緩和期待が高まっています。一方、イランは協議には慎重な姿勢を崩していません。

> **→ 原油価格の下落 → 輸送・製造コスト低下 → 株式市場にプラス**

### 【米国】インテル決算とAI半導体ブーム

インテルの好決算・見通しが半導体セクター全体に強気センチメントをもたらしました。米半導体株指数は18日連続上昇と異例の上昇局面が続いています。来週はマグ7（Apple、Amazon、Microsoftなど）が決算を控えており、AI関連銘柄への注目度が一段と高まります。

### 【米国】FRB議長捜査の終結

米司法省がパウエルFRB議長をめぐる捜査を終了。ケビン・ウォーシュ氏の5月就任に向けた手続きが進展しています。これを受け、米長期金利が低下し株式市場を下支えしました。

### 【日本】為替・財務相発言

片山財務相は円安に関して「断固として強い措置が取れる」「米国と24時間態勢で連携中」と市場をけん制。連休中も米国との連絡を継続する姿勢を示しており、**1ドル＝160円前後が投機的円売りへの防衛ライン**とみられます。

### 【インフレ】消費者物価・景況感

- 日本3月消費者物価: **前年比+1.8%**（2カ月連続で2%下回る）
- 米ミシガン大消費者信頼感（4月確報）: **49.8（過去最低）** ← イラン戦争長期化による物価高懸念
- ブルームバーグのエコノミスト調査: 米国のインフレ予想を上方修正

## 3. 日本株市場の見通し（週明け4月27日以降）

### 予測: **上昇継続、60,000円台前半で底堅い展開**

**強気要因:**
1. **米国株（S&P500・ナスダック）の史上最高値更新** → 投資家センチメント改善
2. **インテル急騰に伴う半導体・電子部品株への連想買い** → 東京エレクトロン、ルネサス等に波及期待
3. **米イラン和平協議進展** → 原油価格下落 → 輸送・航空・素材コスト低下
4. **日経平均先物（夜間）が420円高** → 寄り付きから上昇スタートの可能性
5. **日本企業の好決算ラッシュ**: キーエンス（+12%）、中外製薬（+19%）、日立建機（+9%）等

**弱気要因・リスク:**
1. **円高リスク**: ドル円が159円台前半に落ち着いており、輸出企業の採算悪化懸念
2. **米ミシガン大消費者信頼感が過去最低** → 米国消費の先行き不安
3. **ゴールデンウィーク前の利益確定売り** → 薄商いが警戒感を増幅する可能性
4. **イラン協議不調リスク**: 協議が決裂した場合、中東情勢が再緊迫化
5. **財政審議会**: 北海道新幹線延伸の費用対効果が「中止水準」と指摘 → 建設セクター一部に影響

**注目セクター:**
- **半導体・電子部品**: 東京エレクトロン、信越化学、ルネサスエレクトロニクス（米半導体株高の波及）
- **建設株**: 清水建設（純利益+92%）、矢作建設工業（純利益+49%）等が脚光
- **不動産**: 野村不動産HD（6期連続最高益予想）

## 4. 米国株市場の見通し（週明け4月28日以降）

### 予測: **強気継続、マグ7決算が最大のカタリスト**

**強気要因:**
1. **S&P500・ナスダックが史上最高値圏** → モメンタム継続
2. **来週のマグ7決算** → Apple、Amazon、Microsoftなど主要テック企業が発表予定
3. **AI投資拡大**: Google（Alphabet）がAnthropicに最大400億ドルの追加投資を検討との報道
4. **FRB議長捜査終結** → 金融政策の不透明感が若干低下
5. **半導体株の強気相場継続**: インテル主導の決算相場がナスダックを牽引

**弱気要因・リスク:**
1. **消費者信頼感過去最低（49.8）** → 個人消費見通しに暗雲
2. **インフレ懸念の再燃**: エコノミスト調査でインフレ予想が上方修正
3. **P&G警告**: 原油高が27年6月期に約10億ドル（約1,600億円）の追加コストを見込む
4. **中国リスク**: 中国が米国からの国内テック企業への投資制限へ（技術デカップリングの深化）
5. **Nike人員削減1,400人**: 中国苦戦・競争激化 → 消費セクターの先行き不安

**注目テーマ:**
- **AI・半導体**: Nvidia、SMCI、Broadcom等
- **マグ7決算**: Meta（4/29）、Microsoft（4/30）、Apple・Amazon（5/1）等
- **米イラン協議**: エネルギーセクター（原油価格動向次第）
