## 2. 主要テーマ・注目材料

### (1) 米イラン情勢と和平協議の進展

最大のリスク要因であった米イラン対立について、**トランプ米大統領がウィットコフ特使をパキスタンに派遣**し、イラン外相との直接協議に入った。イランは慎重姿勢を崩していないものの、この動きを市場は好感し、原油相場は1.51%下落。地政学的リスクの一時的な後退が株式市場の上昇を支えた。

- 米国はホルムズ海峡封鎖への警戒から、原油備蓄放出と海軍展開を継続
- 日本政府も石油の国家備蓄を5月1日から順次放出（約20日分）と発表
- 米ミシガン大消費者マインドは過去最低：イラン戦争長期化によるインフレ懸念が根強い

### (2) 半導体・AI株の強気相場継続

- 米半導体株指数は**18日続伸**、ETFへの資金流入が続いている
- **インテルが約24%急伸**。好決算・見通し改善がAI楽観論を一段と強化
- **グーグル（アルファベット）がアンソロピックに最大400億ドル追加投資**との報道が浮上、AIへの大規模投資継続の流れを示す
- AI半導体で台湾・韓国株が時価総額で欧州主要国を逆転
- 中国・DeepSeekが最新AIモデルのプレビュー版を公開し、コスト競争力を示す

### (3) FRB議長人事の進展

**米司法省がパウエルFRB議長に対する捜査を終結**させたことが発表された。これによりウォーシュ氏の5月就任に向けた道が開かれ、市場の不透明感が一部解消。10年債利回りが低下（価格上昇）する形で債券市場が反応し、株式市場にも追い風となった。

### (4) 日本円・為替市場

- **片山財務相**は「投機的動きには断固として強い措置がとれる」と発言を繰り返し、為替介入への準備姿勢を鮮明にしている
- ドル円は現在**159円台**で推移。原油安・米長期金利低下を受けて円はやや反発
- 連休中も日米間で為替に関する24時間連絡体制を維持するとの発言あり

### (5) 日本企業の好決算ラッシュ

| 企業 | 決算内容 |
|---|---|
| キーエンス | 純利益12%増、5年連続最高益（26年3月期） |
| 野村不動産HD | 6期連続最高益（27年3月期4%増予想） |
| 清水建設 | 純利益92%増（26年3月期、上振れ） |
| 中外製薬 | 純利益19%増（血友病薬好調） |

好決算が相次いでおり、日本企業の収益基盤の強さが市場を下支えしている。

### (6) インフレ・物価動向

- 日本の3月消費者物価は前年比**+1.8%**（2カ月連続で2%を下回る）
- 日銀の利上げについて経済学者の見方は分かれ、「円安是正を優先すべき」と「景気下押しリスク」の双方の意見が拮抗
- 米国では2025年度インフレ率が**2.7%**と4年連続で2%超が見込まれ、生活者の不満が高まっている
- 農業資材（JA全農）はナフサ高を受けて仕入れ原価が2〜4割上昇、値上げを実施へ
- 中国のタングステン規制がダーツや釣り具など消費者製品にまで値上がりが波及

## 3. 週明け（4月27日）日本市場の予測

### 総合判断：**やや強気（ポジティブ）**

前日の日経平均史上最高値更新・先物の続伸に加え、NY市場でのS&P500・ナスダック最高値更新、インテル急伸という強い追い風がある。米イラン協議進展による地政学リスク後退も市場心理を改善させており、特に半導体・AI関連銘柄を中心とした上昇基調の継続が予想される。

| 要因 | 方向 | 説明 |
|---|---|---|
| 米国株（S&P・ナスダック最高値） | ↑ 強い追い風 | 特にテック・半導体への波及期待 |
| 日経平均先物 +420円（60,140円） | ↑ 続伸示唆 | 週明け寄り付きはギャップアップの可能性 |
| 米イラン協議進展 | ↑ 地政学リスク後退 | 原油安は製造業・輸送コストにプラス |
| FRB議長捜査終結 | ↑ 不透明感後退 | 米金融政策の先行き安定期待 |
| 円相場（159円台） | → 中立〜やや円高 | 財務相の介入警告で急激なドル高は抑制 |
| 日本企業好決算ラッシュ | ↑ 業績相場の継続 | キーエンス・清水建設等の個別株物色 |
| 米ミシガン消費者マインド過去最低 | ↓ 下押し圧力 | 内需・消費関連には慎重姿勢も残る |
| タングステン・農業資材値上がり | ↓ インフレ懸念 | コスト上昇懸念が製造業の重しに |

### 注目セクター

- **買い目線**：半導体・電子部品、AI関連、建設・不動産、医薬品（中外製薬等）、防衛関連
- **慎重目線**：エネルギー（原油安影響）、消費財（インフレ・コスト高）、輸出企業（円高方向の為替リスク）

## 4. 米国市場の予測（4月28日〜）

### 総合判断：**強気基調維持、ただしリスク注視**

S&P500・ナスダックが最高値圏にある中、来週は**マグニフィセント7を含む主要企業の決算発表が本格化**する。決算内容が市場期待に応えられるかが最大の焦点となる。FRBパウエル体制の終わりとウォーシュ新体制への移行という政策転換点も、不確実性として意識される。

| 要因 | 方向 | 説明 |
|---|---|---|
| S&P500・ナスダック最高値圏 | ↑ 強気継続 | 半導体・AI関連のモメンタム強い |
| インテル急伸による決算サプライズ | ↑ 半導体セクター全体に追い風 | ETFへの継続的な資金流入 |
| 来週のマグ7決算 | → 注目イベント | 期待外れならば調整も |
| 米イラン協議継続 | ↑ リスクオフの緩和 | ただし合意失敗時は急反落リスク |
| FRB議長交代（ウォーシュ5月就任へ） | → 中立〜不透明 | 新体制の政策方針が焦点 |
| 消費者マインド過去最低 | ↓ 内需減速懸念 | 小売・内需関連に逆風 |
| グーグルのアンソロピック400億ドル投資 | ↑ AI投資拡大の継続 | AIクラウド・インフラ関連に好影響 |
| ロシア中銀0.5%利下げ | → 中立 | 新興国・エマージング市場への間接影響 |

## 5. リスクシナリオ

### 上振れシナリオ
- 米イラン間で停戦・和平合意が成立 → 原油急落、リスクオン全面化
- マグ7決算が市場予想を大幅上回る → テック株主導でS&P500が続伸

### 下振れシナリオ
- 米イラン協議が決裂・戦闘再拡大 → 原油急騰、リスクオフ、株式市場急落
- ウォーシュ新FRB議長が利上げ示唆 → 債券売り・株式市場の調整
- ドル円が157円割れへ急速に進む → 輸出株に打撃、日本株に逆風
- 消費者信頼感悪化が実体経済に波及 → 小売・消費関連決算の下振れ

---

## 6. まとめ

2026年4月25日時点で、日米の株式市場は**半導体・AIを中心とした強気相場**が継続している。米イラン協議の進展、インテルをはじめとする好決算、AI投資の拡大が市場の追い風となっている一方、消費者マインドの過去最低水準、インフレの根強さ、為替介入リスク、地政学的不確実性というリスク要因も複数存在する。

週明けの日本市場は**先物・米国株の上昇を受けてポジティブな出発点**が見込まれるが、**円相場の動向とイラン協議の続報**に注目しながら慎重に動向を見守ることが重要である。
