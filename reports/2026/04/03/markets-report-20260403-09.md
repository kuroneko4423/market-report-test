# 本日の株式市場予測レポート（2026年4月3日）

作成日時：2026年4月3日

---

## 総合概況

中東情勢（米イラン紛争・ホルムズ海峡危機）が引き続き市場の最大テーマ。4月2日のトランプ大統領演説が早期停戦への期待を後退させ、WTI原油が一時+11%急騰して113ドル台に達した。エネルギー高による景気押し下げと各中央銀行の利上げバイアスが重なり、リスクオフムードが継続している。なお、欧州・米国は復活祭（イースター）連休中であり、本日から金曜日（グッドフライデー）にかけて欧米市場は休場または短縮取引となる点に注意が必要。

## 日本株式市場

### 予測：**反発、ただし上値は限定的**

**予想レンジ：52,800〜53,500円（日経平均）**

### 根拠

| 項目 | 内容 | 方向 |
|------|------|------|
| シカゴ日経先物（夜間終値） | +830円の53,250円 | ↑強気 |
| 日本株ADR | ほぼ全面安（みずほFG、オリックス等下落） | ↓弱気 |
| 円相場 | 1ドル＝159円55〜65銭（円安継続） | ↑輸出株に追い風 |
| WTI原油 | 一時113ドル台（+11%急騰） | ↓素材・化学株に逆風 |
| 日銀利上げ観測 | 10年JGB利回り2.4%（28年ぶり高水準） | ↓金融コスト上昇 |
| 欧米市場 | イースター休場（流動性低下） | ↓上値追いにくい |

### ポイント別解説

**プラス要因**
- シカゴ先物が前日比+830円（53,250円）で終了しており、寄り付きは反発が見込まれる。
- 日銀短観（3月）で大企業製造業の景況感が4四半期連続改善。国内ファンダメンタルズは底堅い。
- 円安（159円台）は輸出企業（自動車・電機）の業績押し上げ要因。
- バークシャー・ハサウェイが円建て社債発行準備中との報道は、海外投資家の日本株・日本市場への関心を示す。
- 読売新聞が報じた通り、「半導体部品や産業ロボット」への市場関心は依然高く、関連銘柄への選好は続く。

**マイナス要因**
- 米イラン紛争の長期化懸念が根強い。イラン革命防衛隊がアマゾン・オラクル施設への攻撃を宣言しており、地政学リスクが高止まり。
- 原油高（113ドル台）が化学・素材・航空・運輸セクターに直撃。市場予想の純利益を1,000億円超切り下げる見通し（日経）。
- 日本の原油輸入量が3月に前月比3割減、4月以降は一段と減少する見通し。エネルギー確保の不透明感。
- 日銀利上げ期待が10年債利回りを押し上げており（2.4%）、グロース・バリュエーション全般の重し。
- 欧米が休場のため、外国人投資家による積極的な売買は期待しにくく、東京市場は薄商いになりやすい。
- ADRがほぼ全面安のため、個別銘柄の下押し圧力も残る。

**セクター見通し**
- **石油関連（INPEX等）**：原油高恩恵で堅調
- **輸出株（自動車・電機）**：円安追い風で買い優位
- **素材・化学（三菱ケミG等）**：原油高コスト上昇で下落リスク
- **銀行・保険**：金利上昇追い風だが、地政学リスク懸念
- **半導体・ロボット関連**：引き続き選好されやすいが、AI相場の勢い鈍化に注意

## 米国株式市場

### 予測：**もみ合い〜小幅安、イースター前の薄商いで不安定**

**予想：引き続きヘッドライン主導の乱高下、中東報道次第で大きく動く可能性**

### 根拠

| 項目 | 内容 | 方向 |
|------|------|------|
| 前日（4/2）NYダウ | −61ドル（終値）、一時−600ドル超の場面も | 不安定 |
| S&P500 | 前日小幅高で終了 | わずかにプラス |
| VIX（恐怖指数） | 警戒感は前日比低下（みんかぶ報道） | 小康 |
| 原油 | WTI+11%（113ドル台）、エネルギーコスト圧力 | ↓非エネ株に逆風 |
| IMF警告 | 2026年は米利下げ余地限定的、インフレ再燃リスク | ↓株式バリュエーション圧迫 |
| FRB（NY連銀・ダラス連銀） | 現行政策「適切」、エネルギー高は数カ月〜1年で波及 | 据え置き観測 |
| テスラQ1販売 | +6.3%増も市場予想下回り | ↓一部企業業績懸念 |
| イースター休場 | グッドフライデー（4/3）は米株式市場休場 | 流動性低下 |

### ポイント別解説

**プラス要因**
- VIX（恐怖指数）が低下傾向にあり、パニック的な投げ売りには至っていない。
- OPECプラス8カ国がホルムズ海峡再開に備えた増産を検討中との報道は、原油供給懸念の一定の緩和要因。
- スペースX IPO（評価額2兆ドル超目標）など大型案件への期待がIPO市場・テク株センチメントを下支え。
- 米失業保険申請件数が2年ぶり低水準付近に減少しており、雇用市場は底堅い。
- 米貿易赤字（2月）が予想ほど拡大せず。

**マイナス要因**
- グッドフライデーで米国株式市場は**4月3日（本日）が休場**。欧州市場も多くが休場のため、市場参加者が大幅に減少し、流動性が薄い。
- イラン革命防衛隊がアマゾン・オラクルの施設への攻撃を宣言しており、米IT大手への直接的なリスクが市場心理を圧迫。
- IMFが米国の年内利下げ余地は限定的と警告。フランス中銀総裁は「次の行動は利上げの可能性が高い」と発言。
- 原油高（113ドル台）は航空・運輸・素材・化学など幅広いセクターのコスト圧迫要因。
- AI・M7銘柄の時価総額が2.3兆ドル減少しており、AI相場の神通力低下を示唆。
- ブルー・アウルをはじめとするプライベートクレジット市場での解約増加・償還制限はクレジット市場のリスクオフシグナル。

**セクター見通し**（次の取引日＝週明け4月6日以降を視野に）
- **エネルギー**：原油高恩恵で堅調見通し
- **テクノロジー（M7）**：AI相場の勢いに陰りあり、中東リスクも直撃で引き続き不安定
- **素材・化学・航空**：原油高コストで下落圧力継続
- **金融**：金利高止まりは追い風だが、信用リスク懸念が浮上
- **ディフェンシブ（ヘルスケア・公益）**：有事相場でやや選好されやすい

## 主要リスク要因

1. **ホルムズ海峡危機の拡大**：イランが通航料要求・革命防衛隊が米企業施設攻撃。封鎖リスクが高まれば原油は一段高、世界市場にパニック売り波及。
2. **日銀の予想外の利上げ**：10年JGB利回り2.4%が示すように市場の利上げ織り込みが進む。4月会合での決定に要注意。
3. **AI相場の一段の崩壊**：M7時価総額が2.3兆ドル失った後も調整継続なら、ナスダック・日経のハイテク株に連鎖安リスク。
4. **欧米イースター休場による流動性リスク**：薄商いの中でヘッドラインが飛び込むと、値動きが過大になる可能性。
5. **円急騰リスク**：有事の円買いが160円突破後に反転した場合、輸出株への逆風に。

## まとめ

**日本株**：シカゴ先物+830円を受けて寄り付きは53,000円台を回復する見通し。ただし原油高・ADR安・日銀利上げ観測・欧米休場による薄商いが上値を抑える。輸出株・石油関連は買い優位、素材・化学・航空は売り圧力。上値は53,500円が当面の目標。  
**米国株**：本日（4/3）はグッドフライデーで**休場**。週明け4/6以降はイースター明けの市場再開で中東ヘッドラインを再評価する展開。エネルギー高と利下げ期待後退が逆風となる中、ボラティリティ高い相場が続くと予想。
