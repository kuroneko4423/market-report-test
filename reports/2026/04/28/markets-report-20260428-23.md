- **展望レポート**: 2026年度コアCPIを2.8%に上方修正、2027年度も上方修正
- **植田総裁発言の要旨**:
  - 「ビハインド・ザ・カーブに陥らないよう、データを丁寧に点検して次回以降適切に対応」

→ **市場コンセンサス**: 次回利上げは6月または7月会合との見方が拡大

- 中東危機が燃料コスト急騰を通じて企業業績・物価に直撃（中部電力が業績予想見送り）
- ブラックロック: 「イランとの戦争でインフレが長期化し、国債利回りは高止まり」と予測

### その他マクロ

- 日本のコアCPI（特殊要因除く）: 前年比2.5%上昇（2%台の推移が続く）
- 米国のFRBは今週のFOMCで3会合連続据え置き予測

## 4. 翌日以降の日本株式市場予測

### 短期見通し: **やや軟調〜もみ合い**（強気センチメントに頭打ち感）

**強気要因（上昇材料）:**
1. **日銀の据え置きによる金利急騰回避**: 即座の利上げではなく、引き続き緩和環境が続く
2. **自社株買いラッシュ**: 信越化学・富士通・デンソー・アイシン等の大型還元策が需給を支える
3. **内需株・防衛関連の堅調**: 三菱電機等の防衛・防衛関連株が業績好調
4. **グロース250の反発**: 中小型株はセンチメント改善で自律反発の余地あり
5. **円高一服期待**: ドル円159円台で安定すれば輸出株の下落圧力が和らぐ

**弱気要因（下落リスク）:**
1. **6月利上げ観測の強まり**: タカ派的展望レポートが市場のセンチメントを圧迫
2. **日経平均の6万円割れ**: テクニカル上の重要な節目を下回り、売り圧力が続きやすい
3. **半導体・ハイテク株の調整継続**: アドバンテスト・信越化学などへの利益確定売り
4. **連休前の手控え**: ゴールデンウィーク入りに向けてポジション縮小の動き
5. **中東情勢・原油高**: ホルムズ海峡問題が長期化すれば企業コスト増加リスク
6. **長期金利上昇圧力**: 日銀タカ派姿勢を受けて2.5%接近→バリュエーション圧迫

### 総合評価

日銀の「タカ派的据え置き」と展望レポートによる物価上振れ修正は、相場の上値を抑える要因として当面意識される。一方、大型自社株買いや国内企業の堅調な決算が需給面の支えとなり、急落より「もみ合い・上値重い」展開を予想する。

**テクニカルポイント**: 日経平均5万9,000〜6万0,500円がボックス圏と想定。6万円回復なら上値追い、5万9,000円割れなら調整が加速するリスクあり。

### 短期見通し: **中立〜やや強気**（重要イベント通過後の方向感探り）

- 主要企業の決算発表
- 雇用統計などマクロ指標

**強気要因:**
1. FRBの据え置き方針による金融環境の安定
2. インテル社債への旺盛な需要（7.7倍超の応募超過）→企業信用環境の改善

4. ムーディーズによる中国格付け見通し改善→グローバルリスクオフ緩和

**弱気要因:**
1. FRB議長交代をめぐる政治的不透明感
2. OpenAIの売上目標未達→AI投資過熱への懸念

### 総合評価

FRBの据え置き継続が下支えとなり、大崩れは想定しにくい。ただし、個人投資家の強気比率急上昇が過熱感を示すほか、中東情勢・インフレ再加速リスクが上値を制約。FOMC通過後に方向感が定まりやすい展開を予想。
