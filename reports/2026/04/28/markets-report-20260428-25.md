# 株式市場動向レポート｜2026年4月28日

## エグゼクティブサマリー

本日（4月28日）の東京株式市場は、日銀金融政策決定会合の結果と植田総裁の記者会見を受けてタカ派的な解釈が広がり、日経平均が619円安の59,917円と3日ぶりに反落した。一方、シカゴ日経平均先物が61,125円と現物終値を大幅に上回って推移しており、米国市場の動向次第では翌4月29日（昭和の日・祝日前）の需給に影響を与える可能性がある。米国市場では、FRBが3会合連続での金利据え置き観測を維持しており、ダウ平均は小動きが続いている。中東（イラン）情勢とエネルギー価格の高騰が世界市場全体の上値を抑えるリスク要因として引き続き意識されている。

## 1. 本日の日本株式市場（東京市場・大引け）

### 主要指標

| 指標 | 終値 | 前日比 |
|---|---|---|
| 日経平均株価 | 59,917円 | −619円（−1.02%） |
| シカゴ日経平均先物（翌朝参考） | 61,125円 | 現物比 +1,208円 |

### 市場概況

日経平均は前場から軟調な展開が続き、大引けでは3日ぶりに60,000円の節目を割り込んだ。主な下落要因は以下の通り。

**① 日銀金融政策決定会合（タカ派）**
- 政策金利は市場予想通り据え置きを決定したが、植田体制発足以来初めて3名の審議委員が反対票を投じた（早期利上げを主張）
- 展望レポートで2026年度の消費者物価指数（CPI）見通しを大幅に引き上げ（2.8%）
- 生鮮食品・特殊要因を除くコアCPIは前年比+2.5%で伸びが拡大
- 植田総裁は「ホルムズ海峡封鎖が継続しても利上げはあり得る」と踏み込んだ発言を行い、6〜7月の利上げ観測が高まった

**② アドバンテスト株の急落**
- 2027年3月期の業績予想が市場予想を下回り、一時7%超の急落となった
- 半導体関連株全体の重しとなり、前場から利益確定売りが加速した

**③ 円相場の動向**
- 1ドル＝159円台前半で推移。植田総裁の会見中に円が再び下落し、ユーロ円は190円台が視野に入る水準まで上昇した
- ユーロ圏の1年先インフレ期待が4%へ急上昇（イラン戦争の影響）しており、円の構造的弱さが改めて浮き彫りになった

### 個別注目銘柄

| 銘柄 | 動向 |
|---|---|
| アドバンテスト | 一時7%超急落（27年3月期ガイダンスが市場予想下回る） |
| デンソー | 最大約3,136億円の自己株買いTOBを発表、ローム買収提案は取り下げ |
| 富士通 | 1,500億円規模の自己株買いプログラムを発表 |

| 東京ガス | 500億円を上限とした自己株買いを発表 |
| 中部電力 | 中東情勢による調達コスト高騰を理由に通期業績予想を未定に |

### ポジティブ材料

個別銘柄ベースでは約8割の銘柄が上昇。大型株の一部による指数の押し下げが全体像を歪めており、中小型株は堅調な展開となった。りそなHDとJR西日本の資本・業務提携の検討が報じられたことも話題となった。

## 2. 米国株式市場の見通し

### 主要指標（前日終値・先物）

| 指標 | 動向 |
|---|---|
| ダウ平均 | 小動き（小幅安傾向） |
| 米国5年債入札 | 最高落札利回り 3.955% |
| NY原油 | 前日比 +2.09%（5日続伸） |
| NY金 | 前日比 −1.00% |
| インテル社債65億ドル | 応募倍率7.7倍（堅調な信用需要） |

### 注目イベント

今週は米国市場にとって重要イベントが集中している。

- **FOMC（連邦公開市場委員会）**: 3会合連続の金利据え置きが有力視されている。FRBは議長交代を巡る政治的不確実性があるものの、政策スタンスへの影響は限定的との見方が多い
- **雇用統計・経済指標**: 週内に主要指標が相次いで発表される予定であり、市場のボラティリティが高まりやすい
- **地政学リスク（米・イラン）**: 交渉進展への期待が後退したことで原油が5日続伸。米イランの2週間停戦が失効した場合のシナリオに対する警戒感が残る

### 強気・弱気材料の整理

**強気材料**
- インテル65億ドル社債への旺盛な需要（7.7倍超）は信用市場の安定を示唆
- グーグルが米国防総省との大型AIサービス契約を締結、AI関連への期待継続
- FRBの金利据え置き観測がハイテク・グロース株のバリュエーション支持

**弱気材料**
- 中東情勢（イラン戦争）の長期化に伴うエネルギーコスト上昇が企業収益を圧迫
- ブラックロックは「イラン戦争による長期インフレで国債利回りは高止まり」と警告
- ユーロ圏の1年先インフレ期待が4%へ急上昇、グローバルなインフレ再燃リスク
- 米中貿易摩擦：世界経済戦争がトランプ政権の想定外の展開となり、米中首脳会談が不透明に

## 4. 翌4月29日（昭和の日）の市場見通し

4月29日は「昭和の日」で日本市場は休場。東京市場が不在の中、ニューヨーク市場の動向がより注目される。

### 日本株（4月30日以降）の注目点

- **シカゴ日経平均先物が61,125円**と本日終値59,917円を大きく上回っており、米国市場が堅調なら次の東京市場は大幅反発が期待される
- 日銀展望レポートの「タカ派」解釈は一部後退する可能性あり（市場の慣れ・調整）
- 主要企業（トヨタ系サプライヤー等）の決算発表ラッシュが続く
- 生保の2026年度運用方針：長期金利3%を視野に国債投資を慎重化している点が金利動向に影響

### 米国株（本日・4月28日夜間）の見通し

- FOMC結果待ちで上値は限定的も、大崩れのリスクも低い
- 原油高継続によるエネルギー株の堅調さが下支え
- AI・テクノロジー関連は個別決算と地政学リスクへの感応度が高く、ボラティリティに注意

## 5. リスクシナリオ

### 上振れリスク
1. 米イラン停戦・交渉再開→原油急落、リスクオン
2. FOMC後のパウエル（後継議長）発言が予想外のハト派→株高、ドル安
3. 日本企業の大型自己株買い継続が指数の下値を支える

### 下振れリスク
1. ホルムズ海峡の全面封鎖→エネルギー危機深刻化、世界的株安
2. 日銀が市場予想より早期（6月）に利上げ→円高・株安の連鎖
3. 米中首脳会談決裂→グローバルサプライチェーン混乱の加速
