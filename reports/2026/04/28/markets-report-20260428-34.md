## 2. 日本株式市場の予測

### 方向性：**やや軟調〜もみ合い**

#### 弱気材料

| 材料 | 内容 |

| 日銀タカ派シグナル | 3人反対・物価見通し上方修正→早期利上げ観測強まり、グロース株・不動産・金融株への圧力 |
| 円高進行リスク | ドル/円が158〜159円台で推移、さらなる円高は輸出関連株の重荷 |
| 中東情勢の不透明感 | 原油高継続でエネルギーコスト増、製造業・航空・化学セクターに影響 |
| 米ダウの続落 | 前日のNY市場軟調が日本市場への売り圧力に |

#### 強気材料

| 材料 | 内容 |
|------|------|
| 好決算ラッシュ | 富士通・NEC・TDK・三菱電機・大手証券など多数の企業が増益・最高益 |
| AI・半導体関連 | NVIDIAの最高値更新・クアルコム急伸が半導体セクターをサポート |
| 自社株買い増加 | 富士通（1500億円）・デンソー（3136億円）・富士電機（210億円）など |
| 個別株の底堅さ | 本日大引けでも個別株の8割が上昇と内需系は底堅い動き |
| 防衛関連の強さ | 三菱電機が過去最高益見込む（防衛事業拡大） |

#### 注目セクター

- **強い**：防衛・半導体・AI関連、金融（金利上昇メリット）、エネルギー（原油高）
- **弱い**：不動産・公益・高PER成長株（利上げ懸念）、輸出関連（円高）

## 3. 米国株式市場の予測

### 方向性：**中立〜やや軟調（ハイテクは相対的に底堅い）**

#### 主要ポイント

- **FRBの金利据え置き**はほぼ織り込み済みで市場インパクトは限定的
- **中東情勢の膠着**が最大のリスクファクター：原油高→インフレ長期化→長期金利高止まり
- ブラックロックは「イラン戦争によりインフレが長期化し、国債利回りは高止まりする」と警告
- **NVIDIAの最高値更新**がナスダックをサポート、ハイテク大型株への資金集中継続
- 米個人投資家の強気比率が急上昇→**短期的な過熱感・調整リスク**に注意

#### 注目イベント（今後）

- FRB金融政策決定会合（3会合連続据え置き見込み）
- イラン・米国交渉の行方（原油価格を左右）
- 英中銀（30日）：金利据え置き予定

## 7. まとめ

本日の市場は**日銀のタカ派シグナルと中東情勢の不透明感**が主要な重し。日経平均は619円安と反落したが、個別株の8割が上昇しており、好決算銘柄への選別買いが継続している。

翌営業日以降は：
1. **日銀利上げタイミングへの市場の読み**が焦点（植田総裁の「減速が限定的なら利上げ」発言が意識される）
2. **原油・中東情勢の進展**が最大の外部変数
3. **FRBの政策姿勢確認**（据え置き見込みだが声明文のニュアンスに注目）

国内企業の決算発表は概ね好調であり、ファンダメンタルズは堅調。外部環境（地政学リスク・為替）次第の展開が続きそうだ。
