# 株式市場動向予測レポート（2026年4月28日）

## 本日の市場総括

### 日本株式市場

本日の日経平均株価は**3日ぶりに反落し、619円安**で引けた。前場は半導体主力株への利益確定売りが先行し反落して前引けを迎えた後、後場は日本銀行の政策決定会合の結果を受けて下げが加速した。ただし、個別株ベースでは約8割の銘柄が上昇するなど、市場内部の底堅さも確認された。

**要因①：日銀政策決定会合（タカ派のサプライズ）**

本日の最大の材料となったのが日本銀行の金融政策決定会合の結果発表だ。政策金利は据え置きとなったが、その内容が市場にタカ派と受け止められた。

- **植田体制で初めて3名の委員が反対票**（利上げを主張する形での反対とみられ市場がサプライズと受け止めた）
- **26年度の消費者物価見通しを2.8%に大幅上方修正**（生鮮食品・特殊要因除く物価は2.5%上昇と伸びが拡大）
- 植田総裁は「物価上振れリスクが顕在化し、景気の調整が限定的なら利上げ」と明言
- 「ビハインド・ザ・カーブに陥らないよう、さまざまなデータや情報を点検しながら次回以降適切に対応」と発言し、早期利上げを示唆するものと解釈された

**要因②：円相場の急変動**

日銀会合のタカ派的な内容を受けて円買いが進み、ドル円は**一時158円台に上昇**した。午後3時の終値ベースでは159円前半まで戻したものの、上値は依然重い展開。輸出企業の業績懸念が株価の重しとなった。

**要因③：中東情勢の長期化懸念と資源高**

イランとの紛争が長引く中、NY原油先物は一時**100ドルを超え、現在は111ドル台**まで上昇。ナフサ輸入が3月貿易統計で4割減少したことが明らかとなり、石油化学・素材関連企業のコスト高が懸念されている。積水化学は中東起因のコスト増を上期140億円と見込むと発表。

**企業業績は好調（個別株は8割が上昇）**

こうした懸念材料がある一方で、3月期決算発表が本格化しており、好業績報告が相次いだ：

| 企業 | 業績概要 |

| NEC | 26年3月期純利益54%増・最高益、増配 |
| 富士通 | 営業益19%増・11期連続増配、1500億円の自社株買い |
| TDK | 27年3月期純利益15%増・3年連続最高（HDD向け部品好調） |
| 三菱電機 | 今期純利益16.5%増・過去最高見込み（防衛事業拡大が柱） |

- **ナスダック**: ほぼプラス圏で推移

米個人投資家の「強気」比率が**急上昇**しており、相場の過熱感を示唆するシグナルも出ている。

**FRBの動向**: 今週のFOMCでは**3会合連続の金利据え置き**が濃厚。ただし議長交代を巡る政治的な緊張もあり、市場への影響は限定的とみられる。

**中東・原油の影響**: 米イラン交渉が停滞する中、トランプ大統領はイランの提案（ホルムズ海峡に関する提案）に懐疑的な姿勢。原油高はインフレ再燃懸念を高め、米10年債利回りは**4.313%**と高止まりしている。米国債相場の下落は株式市場の上値を抑える可能性がある。

## 明日（4月29日）の相場予測

### 日本株式市場

**予測：やや弱含み～横ばい（下値は限定的）**

日銀の早期利上げ観測が株価の上値を抑える圧力となる。特に金利上昇に敏感な不動産・公益・高配当セクターには売り圧力が続く見込みだ。一方、今期の好決算が続くIT・防衛・機械セクターは個別物色で支えられ、全体的な下落幅は限定的と予想する。

- **日経平均の想定レンジ：60,500円～61,500円**
- シカゴ日経平均先物（61,125円）が示す水準が基準となる

**注目ポイント**:
1. 植田総裁の発言の継続的な影響（利上げ時期の見通し変化）
2. ドル円相場の動向（159～160円台での推移か）
3. 中東情勢の進展（特に米イラン交渉の動き）
4. 引き続く決算発表（4月末の決算集中期）

**セクター別見通し**:
- **強気**: 防衛（三菱電機・IHI等）、IT・DX（NEC・富士通）、HDD部品（TDK）
- **弱気**: 不動産、電力・ガス（金利上昇懸念）、石油化学（原料コスト高）
- **中立**: 自動車（円高圧力 vs 業績改善）、金融（金利上昇は収益プラスだが不良債権懸念）

### 米国株式市場
