# 株式市場予測レポート 2026年4月28日

## 概要

本日（2026年4月28日）の日本・米国株式市場の動向を、各種ニュースソース（日経新聞、ロイター、ブルームバーグ、みんかぶ等）のRSSフィードをもとに分析・予測する。

## 1. 日本株式市場

### 本日の動向

- **日経平均は3日ぶり反落、619円安**（みんかぶ・ロイター）
  - 利益確定売りが広がり、日銀の政策決定会合の結果が追加的な下落要因となった。
  - 日銀は政策金利を据え置いたが、物価見通しの大幅な上方修正（2026年度：2.8%）と、植田総裁の「タカ派的」な発言が市場の利上げ前倒し観測を強めた。

### 日銀金融政策決定会合のポイント

- 政策金利は据え置き（植田体制で初の3名反対）
- 2026年度消費者物価見通しを**2.8%**に大幅引き上げ
- 植田総裁発言：「物価上振れリスクが高まり、景気の調整が限定的なら利上げを検討」
- 「ビハインド・ザ・カーブに陥らないよう、さまざまなデータや情報を点検する」
- 市場はこの発言をタカ派と受け止め、**円買い・株売り**の流れを促進

### 円相場（為替）

- **ドル/円は159円前半**で上値が重い展開
- 日銀会合後の円買いが進み、円高圧力がかかっている
- ユーロ/円相場は190円をうかがう水準でも推移しており、円の構造的弱さが指摘される局面

### 日本株市場の予測

- **短期的にはやや弱含み**の展開が予想される。
- 日銀の早期利上げ観測が強まっており、金利敏感セクター（不動産・公益等）への売り圧力が継続しやすい。
- 一方、防衛・半導体・企業業績良好銘柄（三菱電機、TDK、NEC等）は個別物色が続くと見られる。
- 為替の円高進行（159円台定着→157円方向）が輸出株に対し逆風となる可能性あり。
- エネルギーコスト上昇（原油100ドル超）は製造業コストを押し上げるリスク要因。

## 2. 米国株式市場

### 主要な材料

#### 中東情勢・原油価格

- **NY原油が一時100ドル超え**（日経）
- ブルームバーグによれば**111ドル台まで続伸**（トランプ大統領のイラン提案への対応待ち）
- 米・イラン交渉が停滞しており、ホルムズ海峡の緊張は続く
- LNGタンカーが紛争開始後初めてホルムズ海峡を通過（中国向け）

#### FRB金融政策

- **FRBは3会合連続の金利据え置きを予定**（ブルームバーグ）
- 議長交代を巡る政治的不確実性が市場の関心を一部分散
- インフレ長期化懸念（ブラックロック：国債利回り高止まり継続）

#### 主要企業・セクターニュース

| 企業/事項 | 内容 |
|---|---|
| OpenAI関連株 | 目標未達報道でソフトバンクG・オラクル安い |
| インテル | 社債65億ドルに7.7倍の旺盛な需要（業績回復期待） |
| クアルコム | OpenAIとスマホ向け半導体開発との観測で株価急伸 |
| バークレイズ | 第1四半期増益・投資銀行部門が好調 |
| 韓国株 | 時価総額で英国を抜き世界8位（AI関連株高が押し上げ） |

#### ユーロ圏・国際情勢

- ユーロ圏インフレ期待が大幅上昇（1年先見通し：4%、イラン戦争の影響）
- 中国の格付け見通しが「安定的」に引き上げ（ムーディーズ）
- 中国人民銀が一部銀行に4月の融資拡大を指示（信用急減速を警戒）

### 米国株市場の予測

- **原油高・インフレ長期化懸念**が続くものの、FRBの金利据え置き姿勢が株価を下支えする局面。
- エネルギーセクター（原油高恩恵）は引き続き強含みが予想される。
- テクノロジーセクターはOpenAI関連の一部失望売りはあるものの、半導体（クアルコム等）や AI投資への期待から選別物色は続く見通し。
- インフレ長期化に備え、投資家がヘッジを積み増している点は高値警戒感を示唆。
- 金融株（バークレイズ等）は堅調な決算が続いており、一定の支持材料となる。
- 中東情勢の悪化・ホルムズ海峡の不安定化は、サプライチェーン・エネルギーコストを通じた株式市場全体への下方リスクとして注視が必要。

## 3. 総合分析・注目ポイント

### リスク要因

1. **日銀の早期利上げ観測**：円高進行と株安のリスク、特に輸出株・不動産セクター
2. **中東情勢悪化**：原油高騰の長期化→エネルギーコスト増・インフレ
3. **米国政治リスク**：トランプ政権のイラン政策の不透明感、FRB議長交代問題
4. **中国経済の減速懸念**：人民銀による融資拡大指示は信用収縮のシグナル

### 注目材料

1. **FRB政策決定会合（近日予定）**：3会合連続据え置きが市場コンセンサス
2. **日銀植田総裁の継続発言**：次回利上げ時期の手がかり
3. **中東・イラン情勢**：原油価格と市場センチメントに直結
4. **日本企業の決算発表ラッシュ**：主要企業の業績動向が相場を左右

### セクター別見通し

| セクター | 日本 | 米国 |
|---|---|---|
| エネルギー | 中立〜強気（コスト増だが価格転嫁） | 強気（原油高恩恵） |
| 半導体・テクノロジー | 中立〜強気（TDK、AI関連） | 強気（クアルコム等） |
| 防衛 | 強気（三菱電機等、予算増） | 中立 |
| 金融・証券 | 中立（利上げ期待vs業績好調の綱引き） | 中立〜強気 |
| 輸出・自動車 | 弱気（円高・サプライチェーン混乱） | 中立 |
| 不動産 | 弱気（利上げ観測） | 中立 |

## 4. まとめ

- **日本株**：日銀タカ派姿勢と円高への反応から**短期弱含み**が続く見通し。個別では決算好調銘柄・防衛関連に選別の動き。
- **米国株**：原油高とインフレ懸念が重しになるものの、FRB据え置き姿勢と企業業績（テクノロジー・金融）が下支え。**中東情勢の動向が最大のリスク要因**。
- 共通のリスク：中東戦争の長期化、エネルギーコスト上昇によるグローバルなスタグフレーション圧力
