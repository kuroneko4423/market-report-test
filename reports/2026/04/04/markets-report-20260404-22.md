# 株式市場予測レポート 2026年4月4日

## 概要

本日（2026年4月4日）の株式市場は、米イラン戦争に伴うホルムズ海峡緊張の一時的な緩和を受け、東京市場は大幅反発で引けた。しかし、中東情勢の本質的な不確実性・原油急騰・米サービス業PMI悪化など複合的なリスクが残存しており、来週にかけての相場は上値が重い展開が予想される。

## 3. 市場予測

### 日本株式市場（来週・短期）

**予測：上値の重い展開、方向感は中東情勢次第**

| 要因 | 方向性 |
|---|---|
| 中東情勢（米イラン戦争） | 下押しリスク（高） |
| 原油急騰 | 下押しリスク（コスト増） |
| 半導体・ハイテク株 | 上昇バイアス（米ハイテク連動） |
| 円安（158〜160円） | 輸出株にプラス |
| 春闘賃上げ（日銀利上げ観測） | 金利上昇リスク |
| 米雇用統計（利下げ後退） | 中立〜やや下押し |

- **総合判断：やや弱気**。本日の反発は大幅安の反動によるものであり、持続性は低い。
- 中東情勢が悪化した場合（ホルムズ海峡封鎖、エネルギー施設攻撃拡大）は急落リスク。
- 一方、ホルムズ海峡をめぐる緊張が緩和した場合は半導体・輸出株主導で反発余地。
- 日経平均のレンジは5万2000〜5万4000円程度を想定。

### 米国株式市場（来週・短期）

**予測：不安定な展開継続、ボラティリティ高め**

| 要因 | 方向性 |
|---|---|
| 米イラン戦争の長期化 | 下押しリスク |
| サービス業PMI悪化 | 景気懸念 |
| 雇用統計好調（利下げ後退） | 金利上昇→株には中立〜ネガティブ |
| ハイテク株（半導体） | 下支え要因 |
| トランプ関税 | 不確実性継続 |

- **総合判断：中立〜やや弱気**。グッドフライデーで4日（米国）は休場。
- イラン戦争の動向が最大のリスクファクターであり、発言・事件のたびに急激な上下動が予想される。
- 恐怖指数（VIX）は本日低下したが、再度上昇する可能性あり。

## 4. 注目ポイント

1. **ホルムズ海峡情勢**：米軍戦闘機撃墜・通航料人民元払い案など、情勢は急変しやすい。追加攻撃の有無が週明け相場の最大の焦点。
2. **財務省の為替介入**：ドル円が160円を突破した場合、実弾介入の可能性が高まる。
3. **日銀の次の一手**：賃上げ5%超が続けば4〜5月の利上げ観測が台頭、円高・株安圧力になりうる。
4. **米国サービス業の動向**：PMI悪化が景気後退シグナルになるか否か、次回指標に注目。
5. **社債・地方債市場**：スプレッド拡大（11カ月ぶり）が示すように、信用リスクへの警戒感も高まっている。

---

## 5. まとめ

本日の東京市場は半導体・ハイテク主導で660円高と反発したが、中東情勢（米イラン戦争）・原油急騰・円安進行・米利下げ観測後退という複合リスクは解消されていない。来週の相場は上値の重い展開が予想され、中東関連のヘッドラインに振り回されるボラティリティの高い局面が続く見通しである。投資に際しては中東情勢の変化に細心の注意を払い、エネルギー・防衛関連株と半導体・輸出株のバランスに留意することが重要である。
