# 2026年4月2日（木）株式市場予測レポート

## 概要

本日（4月2日）の日本・米国株式市場は、**米国とイランの軍事衝突の早期終結期待**を主軸としながらも、**UAE（アラブ首長国連邦）のホルムズ海峡への軍事介入示唆**という新たなリスクが交錯する展開が予想される。前日（4月1日）に日経平均が2,675円高と急反発した後、今後の展開は中東情勢の動向に大きく左右される。

---

## 1. 前日（4月1日）の市場動向

### 日本市場
- **日経平均株価**: +2,675円高（一時53,600円台を回復）、5日ぶりの急反騰
- **読売333指数**: +1,958円高（4万7,374円）、3営業日ぶり上昇
- **ドル円**: 158円後半でほぼ横ばい。イラン情勢の進展を見極める様子見姿勢
- **30年国債利回り**: インフレ懸念後退と期初の買い需要で連日大幅低下

### 米国市場（3月31日終値）
- **NYダウ**: +1,125ドル高（イラン戦争の早期終結期待が主因）
- **株価・債券・ドル**: 株高・債券高（利回り低下）・ドル安の構図
- **米ハイイールド債利回り**: 景気減速懸念で上昇傾向、資金流出継続

### 商品市場
- **NY原油**: 96〜103ドル台で乱高下。UAEのホルムズ海峡への軍事介入示唆で103ドル台へ上昇後、停戦期待で急落
- **金（ゴールド）**: 4,755ドル台で高止まり。リスクオフ資産としての需要継続

### 日本経済・金融政策

- **日銀3月短観**: 大企業製造業の景況感が4四半期連続改善。ただし「3カ月後」見通しは3ポイント悪化（中東情勢への懸念）

- **全銀協・加藤新会長**: 日銀利上げ判断は「難しい状態」（中東情勢混乱を理由に）
- **フラット35金利（4月）**: 2.49%と現行制度で過去最大の上げ幅・最高水準更新
- **2月税収**: +5.1%（企業業績好調による法人税収の伸び）
- **ホルムズ危機で電力市場に脚光**: 電力取引が最高値圏

### 米国経済指標

- **ADP民間雇用（3月）**: +6.2万人（予想上回る）、前月も上方修正 → 雇用の底堅さ確認
- **住宅ローン金利（MBA）**: 6.57%（昨年8月以来の高水準）→ 住宅市場の冷え込みリスク

## 3. 本日（4月2日）の市場予測

### 日本株式市場

**見通し: やや強気（ただし中東情勢次第で下振れリスク大）**

前日の大幅反発（+2,675円）の反動売りが懸念されるものの、引き続き米・イラン停戦への期待が下支えとなる展開が見込まれる。ただし、UAEのホルムズ海峡への軍事介入示唆という新リスクが浮上しており、楽観的な見方には慎重さが必要。

**予想レンジ**: 52,000〜54,000円

| 項目 | 予測 | 根拠 |
|---|---|---|
| 方向感 | もみ合い〜小幅続伸 | 停戦期待の持続、ただし前日急騰の反動 |
| 上値抵抗 | 54,000円 | 一時的な達成感と利益確定売り |
| 下値支持 | 52,000円 | 停戦期待の再浮上で押し目買い期待 |
| 注目セクター | AI半導体・電子部品、電力関連、資源関連（防衛関連は戻り売り） | |

**ポジティブ要因**:
- 米・イラン停戦・早期終結期待の持続
- 期初の機関投資家による買い需要（3月末から継続）
- 3月短観の大企業製造業景況感4期連続改善
- AI関連株（アドバンテスト、半導体関連）への継続的な資金流入
- 東証プライムの3月売買代金が191兆円と過去最大（AIブームによる活況）

**ネガティブ要因**:
- UAEのホルムズ海峡介入示唆による地政学リスクの再燃
- 原油価格の不安定な動き（96〜103ドルのボラティリティ）
- 日銀利上げ判断の不透明感（スタグフレーション懸念と利上げ見送りの可能性）
- フラット35金利の過去最高更新による内需への悪影響
- 米ハイイールド債市場の悪化（景気減速懸念の先行指標）

---

### 米国株式市場

**見通し: 中立〜やや弱気（中東情勢の流動性が高く、ボラティリティ継続）**

前日の大幅高（NYダウ+1,125ドル）の後、本日は追加の上昇材料に乏しいなか、イラン情勢の不透明感が続くため、上値の重い展開が想定される。テック株は「安全資産」神話が揺らぎつつあるとの見方も浮上しており、全般的に慎重姿勢が妥当。

**予想レンジ（ダウ）**: 前日比▲500〜+300ドル（もみ合い）

| 項目 | 予測 | 根拠 |
|---|---|---|
| 方向感 | もみ合い〜小幅下落 | 前日急騰後の利確、イラン情勢の流動性 |
| 注目指標 | ADP雇用（発表済み）、週次新規失業保険申請件数 | 雇用市場の底堅さ確認 |
| 注目セクター | AI・半導体（エヌビディア・マーベル）、エネルギー株、航空株（下落リスク）| |

**ポジティブ要因**:
- 米・イラン早期停戦への期待継続
- ADP雇用者数が予想上回る → 景気の底堅さ
- AI・半導体投資の拡大（エヌビディアのマーベル出資、OpenAI過去最大調達）
- JPモルガンCEO（ダイモン氏）の「戦争をうまく完了させることが重要」発言（楽観論の一角）

**ネガティブ要因**:
- UAEの軍事介入示唆 → 原油高の再燃懸念
- トランプ氏の低金利希求がイラン戦争で砕かれるリスク（インフレ継続懸念）
- 住宅ローン金利6.57%（昨年8月以来の高水準）→ 景気冷却リスク
- テック株「安全資産」神話への疑問（株式市場全体の不安定化懸念）
- プライベートクレジット市場での解約急増（金融システムへの潜在リスク）
- NATO脱退をトランプ氏が検討（地政学的不確実性の拡大）

## 6. 総括・投資家へのポイント

1. **中東情勢が全ての鍵**: イラン停戦の進展速度と、UAEの軍事介入の現実化有無が本日の市場を支配する。トランプ氏の「2〜3週間での作戦終結」発言を軸に、停戦期待は根強いが、イランのミサイル攻撃継続・UAEの新リスクが楽観論に冷や水をかけている。

2. **日本株は「離れ小島」リスクに注意**: 前日の日経平均の大幅高が米株や他のアジア株と比較して突出していた点に留意。売買代金の動向が底堅さの真偽を判断する鍵。

3. **AI関連株は引き続き注目**: アドバンテストの転換社債発行、エヌビディアのマーベル出資、OpenAIの過去最大調達など、AI投資拡大の流れは継続。関連銘柄への資金流入は中東リスクの高まる局面でも比較的底堅い。

4. **日銀政策は現状維持方向**: スタグフレーション懸念と原油高が重なる中、日銀の4月利上げ観測は後退。円安バイアスが残りやすい環境。

5. **エネルギー・航空セクターへの影響**: 原油高はエネルギー株に追い風、航空株（ANA・JAL）には燃油サーチャージ引き上げにもかかわらず逆風（コスト増）。
