# 株式市場動向予測レポート（2026年4月27日）

## 概要

本日（2026年4月27日）の日本株式市場では、日経平均株価が取引時間中に初めて6万円台を突破し、史上最高値を更新した。半導体関連株への強力な買い、イランによるホルムズ海峡再開に向けた新提案が市場センチメントを押し上げた。一方で中東情勢の不確実性・週後半に控える日米中央銀行の政策決定会合・ビッグテック決算への警戒感が上値を抑える構図となっている。

---

## 1. 本日の日本市場動向

### 日経平均株価

| 指標 | 動向 |
|------|------|
| 前引け | +868円（約6万584円）→ 後場に一時+1,100円超まで拡大 |
| 終値水準 | 初めて取引時間中に6万円台を記録し、最高値を更新 |
| 寄り付き | SOX指数18連騰を受け半導体株主導で続伸発進 |

**主な上昇要因**
- 米費城半導体株指数（SOX指数）の18連騰が国内半導体関連株の買いを誘発
- イランがホルムズ海峡再開・停戦に向けた新提案を提示（報道）→ 原油リスクプレミアム縮小への期待
- 米国市場でのS&P500最高値更新・台湾株の史上初4万ポイント突破がアジア全体にリスクオン波及
- 国内企業の好業績：大和証券グループ1〜3月期純利益66%増など

**個別株トピック**
- **東京エレクトロン**：米半導体指数連騰を受け株価最高値更新
- **アドバンテスト**：後場の日経平均押し上げの主役
- **キーエンス**：定款変更で自己株取得が可能になるとの思惑でストップ高
- **野村ホールディングス**：2026年3月期最高益を発表も「材料出尽くし」として一時6%安・PBR1倍割れ
- **ローム**：「デンソーが買収提案撤回を検討」と報道され一時16%安（15年ぶり下落率）
- **中外製薬**：1〜3月期増益も通期見通しへの警戒から一時15.7%安
- **はてな**：不正送金で資金流出、ストップ安気配
- **SMC（スーパーマイクロコンピュータ）**：英活動家ファンド・パリサーが自社株買い6,000億円を提案、一時11%高

### 新興市場（グロース250）

換金売り優勢で続落。大型株への資金シフトが鮮明。

### 円相場・債券市場

| 指標 | 動向 |
|------|------|
| ドル円 | 159円台前半（円安基調継続） |
| 長期金利 | 2.450%に上昇（一時2週間ぶりの高水準） |
| 超長期債 | 生保が「先高観乏しい」として円債増加姿勢へ転換の動き |

円相場は「投機筋の円売り積み上がり」と「日銀のタカ派シグナル発信への高い壁」が拮抗し、反転の兆しは見えていない。財務省・片山財務相は米国と24時間態勢で連携し、投機的動きへの介入準備を示唆。

### 商品市場

- **原油**：イランの新提案報道を受け上げ縮小（ホルムズ海峡再開期待で需給懸念後退）
- **金**：買い優勢、地政学リスクヘッジ需要が継続

## 2. 海外市場・グローバル動向

### 米国市場（前週末 4/25 終値）

- **S&P500**：米イラン協議再開への期待を受け最高値更新
- **ナスダック**：インテル好決算・SOX18連騰でハイテク主導で大幅高
- **ダウ平均**：小幅安（ナスダックとの温度差）

### アジア市場

| 市場 | 動向 |
|------|------|
| 台湾株 | 史上初の4万ポイント突破。南亜科技はストップ高 |
| 韓国株 | 最高値更新 |
| 香港株 | 半導体・EV銘柄に買いで続伸 |
| 上海株 | 朝安後半導体買いで上昇に転換。米イラン協議警戒が上値抑制 |

### 中東情勢（最重要テーマ）

イランが「まずホルムズ海峡を再開し停戦、その後に核協議」という新提案を提示（WSJ等報道）。

- イラン外相はロシア当局者と協議へ（外交チャンネル維持）
- フランス大統領もホルムズ海峡再開に注力と表明
- 米国はイラン関連制裁として暗号資産3億4,400万ドル相当を凍結
- ただし第2回米イラン直接協議は実現せず（日本株の米株先物が一時下落した要因）
- ゴールドマン・サックスはホルムズ封鎖長期化を想定し原油相場予想を引き上げ
- シティはブレント原油が150ドルに達する可能性も指摘

→ 「和平への期待」と「交渉破綻リスク」の両側面で市場は神経質な展開が続く。

## 4. 明日（4/28）以降の相場予測

### 日本株市場（東証）

**強気要因（上昇シナリオ）**
1. ビッグテック決算が市場予想を上回る → SOX連騰継続 → 半導体関連株の追い買い
2. イラン・米の交渉進展（ホルムズ海峡一時再開の合意） → 原油安 → リスクオン
3. 日銀が利上げに慎重なスタンスを維持 → 円安継続 → 輸出株・製造業株の下支え
4. 国内企業の好業績発表継続（決算シーズン本格化）

**弱気要因（下落シナリオ）**
1. ビッグテック決算がガイダンスで失望 → ハイテク株急落リスク
2. 中東情勢の急展開（交渉破綻・軍事衝突再燃） → リスクオフ
3. 日銀が予想外のタカ派シグナル → 急速な円高進行 → 輸出株急落
4. 新興株（グロース250）の換金売り圧力継続によるセンチメント悪化

**総合判断**：上昇トレンド継続も上値は重い。6万円〜6万1,000円のレンジでの神経質な動き。
- 半導体関連株：引き続き物色の中心、ビッグテック決算結果が試金石
- 内需株：キーエンスのような自社株買い期待銘柄に個別物色
- 注意銘柄：ローム（買収提案動向）、野村HD（需給懸念）

### 米国株市場

- マグ7決算が注目の的。時価総額2,550兆円超の行方がS&P500の方向性を決定
- FOMCは据え置き予想が大勢だが、スタグフレーション懸念のある声明文が出れば下押し
- イラン情勢は引き続き地政学プレミアムとして市場に残存

### 為替（ドル円）

- 159円台前半の円安継続が基本シナリオ
- 日銀会合でのタカ派サプライズがあれば155円方向への急反転リスク
- 米国のインフレ再燃確認（PCE上振れ）でドル高・円安加速の可能性も
- 財務省の介入警戒水準（160円前後）に近づきつつあり、実際の介入リスクも意識
