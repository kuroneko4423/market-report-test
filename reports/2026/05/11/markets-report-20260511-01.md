## エグゼクティブサマリー

2026年5月11日（月）の日本株式市場は**強気スタートが有力**。前週末の米国市場でS&P500・ナスダックが揃って最高値を更新し、米4月雇用統計が市場予想を大幅上回る結果（非農業部門+11.5万人）となったことが強力な追い風。日本株内では業種別リーダーシップが自動車から半導体・銀行へとシフトし、ソニー（過去最高益1.4兆円）・トヨタ（売上高50兆円超）・任天堂（過去最高売上高2兆円超）など主要企業の決算が良好。一方、リスク要因としてイラン情勢の緊張継続・米消費者マインド指数が過去最低・米FRBの利下げ先送り（ゴールドマン：12月以降）・米中首脳会談（14日、天壇公園）を控えた不透明感が残る。**予想日経225レンジ：63,000〜65,500円**、**米国株は堅調維持見込み**。

|------|------|--------|

| ナスダック | 26,247.08 | +440.88 (+1.71%) ※最高値更新 |
| S&P500 | — | 最高値更新 |
| 米10年債利回り | 4.35% | 低下 |
| USD/JPY | 156.60〜156.70 | 反発（円安方向） |

### 強気材料

1. **4月雇用統計が市場予想を大幅超過**  
   非農業部門雇用者数は前月比+11.5万人（市場予想の約2倍）。AIデータセンター建設・半導体関連が雇用を牽引。労働市場の底堅さが確認されたことで、企業業績への信頼感が持続。

2. **ナスダック・S&P500が揃って過去最高値更新**  
   半導体・ハイテク主導で上昇。AMD・モデルナ・モンスター・ビバレッジが個別で上昇。アルファベットがAI競争で巻き返し、時価総額世界首位も視野に。

3. **インテル・アップルが半導体供給で提携合意（報道）**  
   インテルとアップルの半導体製造暫定合意が米国半導体産業の競争力強化に寄与。インテル株が急伸し、半導体セクター全体への強気センチメントを後押し。

4. **中国4月輸出が急回復（前年比+14%増）**  
   3月の+2.5%から大幅拡大。AI投資関連の貿易が拡大し、グローバル景気の堅調さを示唆。

5. **アルファベットがAI競争で巻き返し**  
   時価総額世界首位の座が視野に。ビッグテック全般への強気センチメントが継続。

### 弱気材料・リスク要因

1. **米消費者マインド指数が過去最低水準**  
   5月速報値が過去最低を記録。インフレの悪影響が家計を圧迫しており、個人消費の下押しリスクが残る。

2. **FRB利下げ先送り**  
   ゴールドマンサックスは利下げ予想を12月・来年3月に後ろ倒し。シカゴ連銀総裁が「あらゆる選択肢を検討」とした一方、ボストン連銀総裁は「金利据え置きを強く支持」。金融引き締め長期化が株式バリュエーションへの圧力となる可能性。

3. **イラン情勢の緊張継続**  
   トランプ氏がイランの和平提案への回答を「全く受け入れられない」と批判。PIMCOのCIOは「イラン戦争に絡み米利上げの可能性を排除せず」と警告。エネルギー価格の上昇リスクが残る。

4. **米中首脳会談（14日）を控えた不透明感**  
   台湾問題での譲歩可能性が焦点。会談結果次第で市場センチメントが大きく揺れる可能性。

5. **トランプ政権の「代替関税」政策への司法判断**  
   国際貿易裁判所の判決が報じられ、対EU車関税引き上げが7月4日まで猶予される状況。貿易摩擦の不確実性が継続。

## 2. 日本株式市場の動向と見通し

### 本日（5月11日）の注目ポイント

| 要素 | 内容 |
|------|------|
| 米国市場の地合い | S&P500・ナスダック最高値更新（強気） |
| 為替 | 1ドル＝156円60〜70銭（前週比：ドル安圧力） |
| 国内企業決算 | ソニー・トヨタ・任天堂が過去最高を更新 |
| 業種別注目 | 半導体・銀行セクターが主役にシフト |

### 強気材料

1. **日本株の主役が半導体・銀行へシフト**  
   「日本株の主役、車から半導体・銀行へ　時価総額10兆円クラブ30社に成長」（日経新聞、5月11日）。自動車依存からの構造転換が鮮明化し、成長セクターが相場を牽引。

2. **主要企業が揃って過去最高業績**  
   - **ソニー**：過去最高益1.4兆円（エンタメ・画像センサー好調）  
   - **トヨタ**：売上高50兆円超（電動車が販売全体の半分）  
   - **任天堂**：売上高2兆円超（スイッチ2が牽引）  
   - **村田機械**：半導体後工程向け大型搬送装置を開発、関連売上高を4400億円規模に拡大予定

3. **実質賃金が3カ月連続プラス**  
   3月実質賃金+1%（減税・エネルギー価格抑制が奏功）。国内個人消費の回復期待が高まる。

4. **法人税収が2割増、税収全体も堅調**  
   3月の国の税収が前年比+3.3%。企業業績拡大が財政面にも好影響。

5. **米雇用統計好結果によるリスクオン地合いの波及**  
   米国市場の強気センチメントが日本株のオープニングを支える。シカゴ日経225先物は上昇を示唆。

### 弱気材料・リスク要因

1. **国の借金が過去最大の1343兆円**  
   財政悪化が長期的な国債需給に与えるリスクへの懸念が残る。

2. **為替介入警戒感**  
   三村財務官が「為替介入に関するIMFルールは回数を制限するものではない」と牽制。円安是正圧力がドル建て資産・輸出株への影響となる可能性。

3. **市場の過熱警戒感**  
   「市場は一瞬で壊れる　『ゆでガエル』日本、株と国債に試練」（日経新聞）という警告報道が示す通り、高値圏での脆弱性への懸念。

4. **イラン情勢・原油価格リスク**  
   日本のエネルギーコストへの影響として、「イラン混迷により日本の所得流出8兆円に」との試算も報告されており、エネルギー輸入コスト上昇リスク。

5. **米中首脳会談の不確実性**  
   14日の米中首脳会談（北京・天壇公園）で台湾問題が焦点となり、結果次第で輸出関連株が振れる可能性。

## 6. 主要リスクシナリオ

### ブルシナリオ（強気）

- 米CPIがインフレ鎮静化を示す → FRB早期利下げ期待が台頭 → 米国株がさらに上昇
- 米中首脳会談が台湾問題での「戦略的曖昧さ」維持で妥結 → リスクオン継続
- イラン情勢が対話路線へ転換 → 原油価格低下で輸入コスト改善
- **日経225：65,000〜67,000円**

### ベースシナリオ（中立）

- 米CPI・米中首脳会談とも無難に通過 → 現状の強気バイアスを維持
- 為替は155〜158円台で推移
- **日経225：63,000〜65,500円**

### ベアシナリオ（弱気）

- 米CPIがインフレ再燃を示す → FRBが利上げを示唆 → 米株急落
- 米中首脳会談が台湾問題で決裂・対立激化 → リスクオフへ転換
- イラン情勢が軍事的衝突に発展 → 原油急騰・エネルギーコスト上昇
- **日経225：60,000〜62,000円**
